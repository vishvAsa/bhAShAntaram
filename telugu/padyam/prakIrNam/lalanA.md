ललना-जनापाङ्ग-,वलना-वसद्-अनङ्ग--,तुलनाभिकाभङ्ग-दोःप्रसङ्गम्  

<details><summary>विस्तारः (द्रष्टुं नोद्यम्)</summary>

वसन्त ऋतुवुलोनि ऒकानॊक रोजु ऎला उन्दो कवि रामराज भूषणुडु वसु-चरित्रलो वर्णिञ्चिन विधं -

ललनाजन = स्त्रीजनं यॊक्क / अपाङ्ग = क्रीगण्टि चूपुल / वलन = चलनंलो / आवसत् = निवसिञ्चे / अनङ्ग = मन्मथुनितो / तुलन = समानमैनवारैन / अभिक = कामुकुल यॊक्क / आभङ्ग = अन्तरायं लेनि / दोःप्रसङ्गमु = कौगिलिन्तल मुच्चट्लु कलदी
</details>

अलसानिल-विलोल-,दळ--सासव-रसाल--,फल-सादर-शुकालपन-विशालम्  

<details><summary>विस्तारः (द्रष्टुं नोद्यम्)</summary>

अलस = मॆल्लगा वीचे / अनिल = गालिचेत ( मन्दमारुतं चेत ) / विलोल = बागा कदुलुतुन्न / दळ = चिगुराकुलु कलवी / स + आसव = मकरन्दन्तो कूडिनवी अयिन / रसाल = तिय्य मामिडि चॆट्ल / सादर = आदरन्तो कूडिन / शुक = चिलुकल / आलापन = आलापाल चेत / विशालमु = विस्तारमैनदी
</details>

अलिनी-गरुद्-अनीक--मलिनी-कृत-धुनी+++(=नदी)+++-कमलिनी--सुखित-कोक-कुल-वधूकम्  

<details><summary>विस्तारः (द्रष्टुं नोद्यम्)</summary>

अलिनी = आड तुम्मॆदल / गरुत् = रॆक्कल / अनीक = समूहं चेत / मलिनीकृत = नल्लगा चेयबडिन / धुनी = नदी सम्बन्धमैन / कमलिनी = तामर तीगल्लो / सुखित = सुखङ्गा उन्न / कोककुल वधूकमु = चक्रवाक स्त्रीलु कलदी
</details>

अतिकान्त-सलतान्त-लतिकान्तर--नितान्त-रति-कान्त-रण--तान्त-सुतनु-कान्तम्  

<details><summary>विस्तारः (द्रष्टुं नोद्यम्)</summary>

अतिकान्त = मिक्किलि मनोहरालैन / सलतान्त = पूलतो कूडिन / लतिक = तीगल / अन्तर = लोपल / नितान्त = ऎडतॆगनि / रतिकान्त = रति भर्त ऐन मन्मथुनि / रण = युद्धंलो / तान्त = अलसिन / सुतनुकान्तमु = स्त्री पुरुषुलु कलदी
</details>

अकृतकामोद-कुरवक+अविकल-वकुल-  
मुकुल-सकल--वनान्त--प्रमोद-चलित-  
कलित-कल-कण्ठ-कुल--कण्ठ-काकली+++(=मधुरध्वनि)+++-वि-  
भासुरमु वॊल्चु मधुमास-वासरम्बु!  

<details><summary>विस्तारः (द्रष्टुं नोद्यम्)</summary>

अकृतक = सहजमैन / आमोद = परिमळं कल / कुरवक = गोरण्ट पुव्वुलु कलवी / अविकल = विच्चिन्नं कानि / वकुळ = पॊगड चॆट्ल / मुकुळ = मॊग्गलु कलवी अयिन / सकल = समस्त / वनान्त = वन मध्यंलो / प्रमोद = सन्तोषन्तो / चलित = सञ्चरिस्तुन्न / कलित = ऒप्पुगा उन्न / कलकण्ट = कोयिलल / कुल = समूहं यॊक्क / कण्ट काकली = कण्ठन्नुण्डि वॆलुवडे मधुर ध्वनि चेत / भासुरमु = प्रकाशिस्तुन्न / मधुमास वसरम्बु = वसन्त ऋतुवुलोनि ऒक दिनं / पॊल्चुन् = विलसिल्लुतुन्नदि.

वसन्त ऋतुवुलोनि ऒकानॊक रोजु स्त्रील कडगण्टि चूपुलु शृङ्गार रस प्रेरितालू मन्मथोद्दीपकालू अय्यायि. कामुकुलु स्वेच्चगा क्रीडिञ्चारु. तिय्य मामिडि चिगुरिञ्चि पण्ड्लु पण्डि फलरसालु दॊरिकायि. आ पण्ड्लनु तिनि चिलुकलु पलिकायि. नदुललो तमर तीगलु बाग पूचायि. वाटिनि तुम्मॆदलु गुम्पुगा कम्मुकॊन्नायि. आ तमर तीगल्लो चक्रवाकालु सुखङ्गा उन्नायि. पू पॊदरिण्ड्ललो स्त्री पुरुषुलु विनोदिस्तुन्नारु. गोरण्टलु पूचायि. पॊगडलु मॊग्ग तॊडिगायि. कोयिललु मधुर ध्वनुलु चेसायि.
</details>
