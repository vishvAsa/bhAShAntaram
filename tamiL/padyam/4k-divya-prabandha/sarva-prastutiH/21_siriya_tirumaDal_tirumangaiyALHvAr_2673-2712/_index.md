


೧. ಕಾರಾರ್ ವರೈಕ್ಕೊಙೈ ಕಣ್ಣಾರ್ ಕಡಲುಡುಕ್ಕೈ,

ಶೀರಾರ್ ಶುಡರ್ ಶ್ಶುಟ್ಟಿ ಶೆಙ್ಗಲುೞಿ ಪ್ಪೇರಾಟೃ

೨. ಪ್ಪೇರಾರಮಾರ್ ಬಿಱ್ ಪೆರುಮಾ ಮಱೈಕ್ಕೂನ್ದಲ್,

ನೀರಾರ ವೇಲಿ ನಿಲಮಙ್ಗೈಯೆನ್ನುಮಿ

೩. ಪ್ಪಾರೋರ್ಶೊಲಪ್ಪಟ್ಟ ಮುನ್ ಱನ್ ಱೇ, ಅಮ್ಮುನ್ ಱುಮ್

ಆರಾಯಿಲ್ ತಾನೇ ಅಱಮ್ಪೊರುಳಿನ್ಬಮೆನ್ ಱು – 



ಕಾರ್ ಆರ್ = ಕಾರ್ಮುಗಿಲು ತುಂಬಿರುವ, ವರೈ = ಬೆಟ್ಟಗಳನ್ನು, ಕೊಙ್ಗೈ = ಮೊಲೆಗಳಾಗಿಯೂ, ಕಣ್ ಆರ್ = ಕಣ್ಣುತುಂಬುವಂಥ, ಕಡಲ್ = ಕಡಲನ್ನು, ಉಡುಕ್ಕೈ = ಉಡುಗೆಯಾಗಿಯೂ, ಶೀರ್ ಆರ್ = ಪ್ರಕಾಶವು ತುಂಬಿರುವ \(ಜ್ಯೋತಿಯೆಂಬ ಐಶ್ವರ್ಯವನ್ನು ತುಂಬಿಕೊಂಡಿರುವ\), ಶುಡರ್ = ಸೂರ್ಯ ನನ್ನ ಶುಟ್ಟಿ = ತಲೆಬೊಟ್ಟಾಗಿಯೂ \(ಆಭರಣವಾಗಿಯೂ\), ಶೆಮ್ = ಕೆಂಪನೆಯ, ಕಲುೞಿ = ಕದಡಿದ ಪ್ರವಾಹದ, ಪೇರ್ = ದೊಡ್ಡ, ಆಱು = ನದಿಯ \(ನದಿಗಳ\) ಪೇರ್ ಆರಮ್ = ದೊಡ್ಡ ಹಾರವನ್ನುಳ್ಳ, ಮಾರ್ ಬನ್ = ಎದೆಯನ್ನುಳ್ಳವಳಾಗಿಯೂ ಪೆರುಮಾ ಮೞೈ = ಬಿರುಸು ಮಳೆಯೆಂಬ, ಕೂನ್ದಲ್ = ಕೂದಲನ್ನುಳ್ಳವಳಾಗಿಯೂ, ನೀರ್ ಆರಮ್ = ಸುತ್ತುಗಟ್ಟಿರುವ ನೀರಿನ, ವೇಲಿ = ರಕ್ಷಣೆಯನ್ನುಳ್ಳವಳಾಗಿಯೂ, ಇರುವ, ನಿಲಮ್ ಮಙ್ಗೈ = ಭೂದೇವಿ, ಎನ್ನುಮ್ = ಎಂಬ, ಇ ಪಾರೋರ್ = ಈಭೂಲೋಕದವರಿಂದ, ಶೊಲಪ್ಪಟ್ಟ = ಹೇಳಲ್ಪಟ್ಟವು, ಮೂನ್ ಱು ಅನ್ ಱೇ = ಮೂರೇ ಅಲ್ಲವೇ, ಅಮುಣ್ ಱುಮ್ = ಆ ಮೂರೂ, ಆರಾಯಿಲ್ ತಾನೇ = ಗಮನಿಸಿ ನೋಡಿದರೆ, ಅಱಮ್, ಪೊರುಳ್, ಇನ್ಬಮ್ = ಧರ್ಮ, ಅರ್ಥ, ಕಾಮ, ಎನ್ ಱು = ಎಂದು.



ಕಾರ್ಮುಗಿಲುಗಳಿಂದ ತುಂಬಿರುವ ಬೆಟ್ಟಗಳನ್ನು ಮೊಲೆಗಳನ್ನಾಗಿಯೂ ಕಣ್ಣುತುಂಬುವಂಥ \(ವಿಸ್ತಾರವಾದ\) ಕಡಲನ್ನು ಉಡುಗೆಯಾಗಿಯೂ, ಜ್ಯೋತಿಯೆಂಬ \(ಐಶ್ವರ್ಯದಿಂದ ತುಂಬಿರುವ ಸೂರ್ಯನನ್ನು ತಲೆಬೊಟ್ಟನ್ನಾಗಿಯೂ, ಕೆಂಪಗೆ ಕದಡಿದ ಪ್ರವಾಹವುಳ್ಳ ದೊಡ್ಡ ನದಿಗಳನ್ನು ಹಾರಗಳ ಎದೆಯನ್ನಾಗಿಯೂ, ಬಿರುಸುಮಳೆಯೆಂಬ ಕೂದಲನ್ನಾಗಿಯೂ, ಸುತ್ತುಗಟ್ಟಿರುವ ನೀರಿನ ರಕ್ಷಣೆಯನ್ನಾಗಿಯೂ ಉಳ್ಳ ಭೂದೇವಿ ಎಂಬ ಈ ಭೂಲೋಕದವರಿಂದ ಹೇಳಲ್ಪಡುವುದು \(ಪುರುಷಾರ್ಥಗಳು\) ಮೂರೇ ಅಲ್ಲವೇ? ಆ ಮೂರು ಎಂದರೆ, ಧರ್ಮ, ಅರ್ಥ, ಕಾಮ ಎಂಬವು.



’ಮಡಲ್’ ಎಂದರೆ, ಪ್ರೇಮಗೀತೆ. ವಿರಹಿಯಾದವಳು ತನ್ನ ಪತಿಯನ್ನು ಕುರಿತು, ತನ್ನ ಮನೋಗತವೆಂಬ ಅಂತರಂಗ ಪ್ರೇಮವನ್ನು ತೋಡಿಕೊಳ್ಳುವ ಗೀತೆ – ವಿರಹಗೀತೆ.



ತಿರುಮಂಗೈ ಆಳ್ವಾರರು ಕವಿ ಶ್ರೇಷ್ಠರೆನಿಸಿ ಬರೆದಿರುವ ಪ್ರೇಮಗೀತೆಗಳು ಎರಡು – ಅವುಗಳಲ್ಲಿ ಇದು ಚಿಕ್ಕದು, ಆದ್ದರಿಂದ ಇದಕ್ಕೆ “ಶಿಱಿಯ ಮಡಲ್ “ ಎಂದು ಹೆಸರು. 



’ಪುರುಷ’ ಎಂದರೆ ಒಬ್ಬನೇ\! ಅವನೇ ಪರಮಪುರುಷನಾದ ಸರ್ವೇಶ್ವರ. ಆ ಸರ್ವೇಶ್ವರನಲ್ಲಿ ಅನುರಕ್ತರಾಗಿರುವ ಜಗತ್ತಿನ ಎಲ್ಲ ಜೀವಿಗಳೂ, ಗಂಡಸು, ಹೆಂಗಸು ಎಂಬ ಭೇದವಿಲ್ಲದಂತೆ. ಆ ಅರ್ಥದಲ್ಲಿ ಸ್ತ್ರೀಯರೇ\! ಅವನಿಂದ ಬೇರ್ಪಟ್ಟು ವಿರಹವೇದನೆಯನ್ನು ಅನುಭವಿಸುತ್ತಾ, ಅವನನ್ನು ಸೇರಲೇಬೇಕೆಂಬ ಉತ್ಕಟವಾದ ಆಶೆಯನ್ನು ಪಡೆದು, ಅವನಿಗಾಗಿ ಕಾತರಪಡುತ್ತಾ, ಇರುವವರು ಜಗತ್ತಿನವರು ಆಧ್ಯಾತ್ಮಿಕವಾಗಿ ಇದು ಆತ್ಮ-ಪರಮಾತ್ಮ ಸಂಬಂಧವನ್ನು ಸೂಚಿಸುವುದೆಂಬುದು ಖಚಿತ.



’ಶಿಱಿಯ ಮಡಲ್’ ಪ್ರಬಂಧದ ನಾಯಕಿಯೇ ತಿರುಮಂಗೈ ಆಳ್ವಾರರು. ಅವರು ತಮ್ಮ ಪ್ರಿಯತಮನಾದ ಭಗವಂತನನ್ನು ಕೂಡಿಕೊಳ್ಳುವುದಕ್ಕಾಗಿ ಮಾಡುವ ಪ್ರಯತ್ನವೇ ಇಲ್ಲಿ ವಿವರಿಸಲಾಗಿರುವುದು.



ಪುರುಷಾರ್ಥಗಳು ನಾಲ್ಕು ಎಂತಲೂ, ಅವು ಧರ್ಮ, ಅರ್ಥ, ಕಾಮ, ಮೋಕ್ಷ ಎಂತಲೂ ವೇದಶಾಸ್ತ್ರಗಳಿಂದ ತಿಳಿಯ ಬಂದಿದೆ. ಆದರೆ, ಜಗತ್ತಿನ ಜನ ಆಶಿಸುವುದು ಮೂರನೇ, ಎಂದರೆ ಧರ್ಮ, ಅರ್ಥ, ಕಾಮಗಳನ್ನೇ \(ಅಱಮ್, ಪೊರುಳ್, ಇನ್ಬಮ್\) ಪರಿಶುದ್ಧವಾದ ಕಾಮಮಾರ್ಗವನ್ನು ಅನುಸರಿಸಿದರೆ, ಪರಪಪುರುಷನನ್ನು ಪಡೆದುಕೊಳ್ಳಲು ಸುಲಭ ಎಂಬುದನ್ನು ಇಲ್ಲಿ ಪ್ರತಿಸಾಧಿಸಲಾಗುತ್ತದೆ. ಈ ಕಾರಣದಿಂದ ನವವಿಧ ಭಕ್ತಿಭಾವಗಳಲ್ಲಿ ಒಂದಾದ ನಾಯಕ, ನಾಯಕೀಭಾವವನ್ನುಇಲ್ಲಿ ಆರಿಸಿಕೊಳ್ಳಲಾಗಿದೆ. 



’ಶಿಱಿಯ ಮಡಲ್’ ಎಂಬುದು ದೀರ್ಘಆದ ಒಂದೇ ಹಾದು. ಆದರೆ, ತಿಳಿವಳಿಕೆಗೆಅನುಕೂಲವಾಗುವಂತೆ ಅದನ್ನು ಎರಡುಸಾಲುಗಳ ನಲವತ್ತು ಪಾಶುರಗಳನ್ನಾಗಿ ವಿಂಗಡಿಸಿಕೊಂಡಿದ್ದಾರೆ. ಅರ್ಥಕ್ಕೆ ಅನುಕೂಲಿಸುವಂತೆ, ಹಿರಿಯರು ಅನುಸರಿಸಿರುವ ಹಾಗೆಯೇ, ಅವುಗಳನ್ನು ಜೋಡಿಸಿ, ಅಥವಾ ಬಿಡಿಸಿ ತೆಗೆದುಕೊಳ್ಳಲಾಗಿದೆ. ಪ್ರಾರಂಭದ ಮೊದಲ ಮೂರುಪಾಶುರಗಳನ್ನು ಒಟ್ಟುಗೂಡಿಸಲಾಗಿದೆ. 



ಈ ಮೊದಲ ಮೂರು ಪಾಶುರಗಳಲ್ಲಿ – ಎರಡು ವಿಷಯಗಳು ಕಂಡು ಬರುತ್ತವೆ: ಒಂದು, ಭೂದೇವಿಯನ್ನು ವರ್ಣಿಸುವ ರೂಪಕ. ಎರಡನೆಯದು ಭೂಲೋಕದ ಜನರ ಬಾಳಿನ ಬಗೆ. ಪ್ರಕೃತಿ ಸಹಜವಾದ ವಿಷಯಗಳನ್ನು ಎತ್ತಿಕೊಂಡು, ಭೂದೇವಿಯ ಸೊಬಗನ್ನು ವರ್ಣಿಸಲಾಗಿದೆ. ಭೂದೇವಿಯೂ ತಾನೂ ಸ್ತ್ರೀಯರೇ ಆಗಿದ್ದರೂ, ಭಗವಂತನ ನಿತ್ಯಾನಪಾಯಿನಿಯಾದ ಆಕೆಗೂ ತನಗೂ ಎಲ್ಲಿಯ ಸಾಟಿ ಎಂಬುದು ನಾಯಕಿಯ ಮನಸ್ಸಿನಲ್ಲಿರಬಹುದೇನೋ\!



ಆಳ್ವಾರರು ಹೇಳುತ್ತಾಳೆ: ಪರಮಪುರುಷನಾದ ಭಗವಂತನನ್ನು ಎಡೆಬಿಡದಂತೆ ಆಶ್ರಯಿಸಿರುವ ಭೂದೇವಿಯ ಮಕ್ಕಳಾದ ಜನರಿಗೆ ಕಂಡುಬರುವ ಪುರುಷಾರ್ಥಗಳು ಮೂರೇ – ಅಱಮ್ \(ಧರ್ಮ\), ಒರುಳ್ \(ಅರ್ಥ\), ಮತ್ತು ಇನ್ಬಮ್ \(ಕಾಮ\) ಎಂಬವು. ಅವುಗಳ ಮೂಲಕ ಅವರು ತಮ್ಮ ಜೀವನದ ಗುರಿಯನ್ನು ಮುಟ್ಟಿದೆನೆನ್ನುತ್ತಾರೆಂದು ತೋರುತ್ತದೆ. 



೪. ಆರಾ ರಿವಟ್ರಿನಿಡೈಯದನೈ ಯೆಯ್ದುನಾರ್,

ಶೀರಾರಿರು ಕಲೈಯು ಮೆಯ್ದುವಾರ್, ಶಿಕ್ಕನ ಮಟ್ರು



೫. ಆರಾನು ಮುಣ್ಡಿನ್ಬಾರೆನ್ಬದು ತಾನದುವುಮ್,

ಓರಾಮೈಯನ್ ಱೇ ಉಲಹತ್ತಾರ್ ಶೊಲ್ಲುಂಜೊಲ್,



೬. ಓರಾಮೈ ಯಾಮಾ ಱದುವುರೈಕ್ಕೇನ್ ಕೇಳಾಮೇ,

ಕಾರಾರ್ ಪುರವಿಯೋೞ್ ಪೂಣ್ಡ ತನಿಯಾೞಿ



೭. ತ್ತೇರಾರ್ ನಿಱೈಕದಿರೋನ್ ಮಣ್ಡಲತ್ತೈ ಕೀಣ್ಡು ಪುಕ್ಕೂ,

ಆರಾವಮದುಙ್ಗೆಯ್ದಿ, ಅದಿಲ್ ನಿನ್ ಱುಮ್



೮. ವಾರಾದೊೞಿವದೊನ್ ಱುಣ್ಡೇ, ಅದು ನಿಱ್ ಕ

ಏರಾರ್ ಮುಯಲ್ ವಿಟ್ಟು ಕ್ಕಾಕ್ಕೈಪ್ಪಿನ್ ಪೋವದೇ – 



ಆರ್ ಆರ್ = ಯಾರು ಯಾರು, ಇವಟ್ರಿನಿಡೈ= ಇವುಗಳಲ್ಲಿ \(ಈ ಮೂರು ಪುರುಷಾರ್ಥಗಳಲ್ಲಿ\), ಅದನೈ = ಅದನ್ನು \(ಕಾಮಪುರುಷಾರ್ಥವನ್ನು\), ಎಯ್ದುವಾರ್ = ಪಡೆಯುತ್ತಾರೆ. \(ಅವರು\), ಶೀರ್ ಆರ್ = ಹಿರಿಮೆಯಿಂದ ತುಂಬಿರುವ, ಇರು ಕಲೈಯುಮ್ = ಇನ್ನೆರಡು ಪುರುಷಾರ್ಥಗಳನ್ನೂ \(ಧರ್ಮ, ಅರ್ಥಗಳನ್ನೂ\), ಎಯ್ದುವಾರ್ = ಪಡೆಯುತ್ತಾರೆ. ಶಿಕ್ಕಿನ = ಕಠಿಣವಾದ \(ತೊಡಕಿನಿಂದ ಕೂಡಿದ\), ಮಟ್ರು = ಮತ್ತೊಂದು \(ಪುರುಷಾರ್ಥ\), ಆರಾನುಮ್ = ಯಾರಾದರೂ, ಉಣ್ಡು ಎನ್ಬಾರ್ = ಇದೆ ಎನ್ನುವರು, ಎನ್ ಬದುತಾನ್ = ಎಂಬುದೂ ಸಹ, ಅದುವುಮ್ = ಅದೂ ಸಹ, ಓರ್ ಮೈ ಅನ್ಱೇ = ಪರೀಕ್ಷಿಸತಕ್ಕದ್ದೇ ಅಲ್ಲವೇ, ಉಲಹತ್ತಾರ್ = ಲೋಕದವರು, ಶೊಲ್ಲುಮ್ ಶೊಲ್ = ಹೇಳುವ ಮಾತು, ಓರಾಮೈಆಮ್ ಆಱು = ಪರಿಶೀಲಿಸುವ ಹಾಗೆ \(ರೀತಿಯಲ್ಲಿ\), ಅದು = ಅದನ್ನು, ಉರೈಕ್ಕೇನ್ = ಹೇಳುತ್ತೇನೆ \(ವಿವರಿಸುತ್ತೇನೆ\), ಕೇಳ್ ಆಮೇ = ಕೇಳತಕ್ಕದ್ದೇ ಅಲ್ಲವೇ, ಕಾರ್ ಆರ್ = ಕಾರ್ಮುಗಿಲು ತುಂಬಿರುವ ಸ್ಥಳದಲ್ಲಿ \(ಗಗನದಲ್ಲಿ\), ಪುರವಿ ಏೞ್ = ಏಳು ಕುದುರೆಗಳನ್ನು, ಪೂಣ್ಡ = ಹೂಡಿರುವ, ಒಂಟಿಯಾದ, ಆೞಿ = ಚಕ್ರದ, ತೇರ್ ಆರ್ = ತೇರಿನ ತುಂಬ, ನಿಱೈ = ತುಂಬಿರುವ, ಕದಿರೋನ್ = ಸೂರ್ಯನ ಮಂಡಲತ್ತೈ = ಮಂಡಲವನ್ನು, ಕೀಣ್ಡು = ಸೀಳಿಕೊಂಡು, ಪುಕ್ಕು = ಪ್ರವೇಶಿಸಿ, ಆರಾವಮುದಮ್ ಎಯ್ದಿ = ಪರಮಪುರುಷನನ್ನು ಸೇರಿ, ಅದಿಲ್ ನಿನ್ ಱುಮ್ = ಅದರಿಂದ \(ಅಲ್ಲಿಂದ\), ವಾರಾದು = ಬರದಂತೆ, ಒೞಿವದು = ಇದ್ದುಬಿಡುವುದು, ಒನ್ ಱು ಉಣ್ಡೇ = ಯಾವುದಾದರೂ ಉಂಟೇ, ಅದು ನಿಱ್ ಕ = ಅದು ಹಾಗಿರಲಿ, ಏರ್ ಆರ್ = ಸೊಬಗು ತುಂಬಿರುವ, ಮುಯಲ್ = ಮೊಲವನ್ನು, ವಿಟ್ಟು = ಬಿಟ್ಟು, ಕಾಕ್ಕೈಪಿನ್ ಪೋವದೇ = ಕಾಗೆಯ ಹಿಂದೆ ಹೋಗುವದೇ? 



ಇವುಗಳಲ್ಲಿ \(ಈ ಮೂರು ಪುರುಷಾರ್ಥಗಳಲ್ಲಿ\) ಯಾರು ಯಾರು ಅದನ್ನು \(ಕಾಮಪುರುಷಾರ್ಥವನ್ನು\) ಪಡೆದುಕೊಳ್ಳುತ್ತಾರೋ ಅವರು ಹಿರಿಮೆಯಿಂದ ಕೂಡಿದ ಇನ್ನೆರಡು ಪುರುಷಾರ್ಥಗಳನ್ನೂ ಪಡೆಯುತ್ತಾರೆ. ತೊಡಕಿನಿಂದ ಕೂಡಿದ ಕಠಿಣವಾದ ಇನ್ನೊಂದು ಪುರುಷಾರ್ಥವಿದೆ\(ಮೋಕ್ಷ\) ಎಂದು ಯಾರಾದರೂ ಹೇಳುವುದು ಸಹಜವಾದರೂ, ಲೋಕದವರು ಹೇಳುವ ಅದನ್ನೂ ಪರೀಕ್ಷಿಸತಕ್ಕದ್ದೇ ಅಲ್ಲವೇ? ಪರಿಶೀಲಿಸುವ ರೀತಿಯಲ್ಲಿ ಅದನ್ನು ವಿವರಿಸಿ ಹೇಳುತ್ತೇನೆ. ಅದನ್ನೂ ಕೇಳತಕ್ಕದ್ದೇ ಅಲ್ಲವೇ? ಕಾರ್ಮುಗಿಲು ತುಂಬುವ ಗಗನದಲ್ಲಿ ಏಳು ಕುದುರೆಗಳನ್ನು ಹೂಡಿದ ಒಂಟಿ ಚಕ್ರದ ತೇರಿನ ತುಂಬ ತುಂಬಿರುವ ಸೂರ್ಯಮಂಡಲವನ್ನು ಸೀಳಿಕೊಂಡು, ಪ್ರವೇಶಿಸಿ, ಪರಮಪುರುಷನನ್ನು ಸೇರಿ, ಅಲ್ಲಿಂದ ಬರದಂತೆ ಇದ್ದುಬಿಡುವುದು ಯಾವುದಾದರೂ ಉಂಟೇ? ಅದು ಹಾಗಿರಲಿ. ಸೊಬಗು ತುಂಬಿದ ಮೊಲವನ್ನು ಬಿಟ್ಟು ಕಾಗೆಯನ್ನು ಹಿಂಬಾಲಿಸಿ ಹೋಗುವುದೇ? 



ಭೂಲೋಕದ ಗೃಹಸ್ಥನು ಅನುಸರಿಸುವ ಧರ್ಮ, ಅರ್ಥ, ಕಾಮಗಳೆಂಬ ಮೂರು ಪುರುಷಾರ್ಥಗಳಲ್ಲಿ, ತಕ್ಕ ರೀತಿಯಲ್ಲಿ ಕಾಮಪುರುಷಾರ್ಥವನ್ನು ಅನುಸರಿಸುವುದರಿಂದ ಅದು ಮಿಕ್ಕೆರಡು ಪುರುಷಾರ್ಥಗಳನ್ನೂ ಅನುಸರಿಸುವಂತೆ ಮಾಡುತ್ತದೆ ಎಂದು ಇಲ್ಲಿ ಹೇಳಲಾಗಿದೆ. ಇದಕ್ಕೆ ಸ್ವಲ್ಪ ವಿವರಣೆಯ ರೂಪವಾಗಿ ಒಂದೆರಡು ಮಾತುಗಳನ್ನು ಹೇಳಬೇಕಾಗುತ್ತದೆ. 



ಭೂಲೋಕದ ಗೃಹಸ್ಥನಿಗೆ, ಅವನ ಬಾಳ್ವೆಯಲ್ಲಿ ಎರಡು ಮಾರ್ಗಗಳು ಕಂಡು ಬರುತ್ತವೆ; ಒಂದು ಪ್ರವೃತ್ತಿಧರ್ಮ \(ಮಾರ್ಗ\), ಮತ್ತೊಂದು ನಿವೃತ್ತಿ ಮಾರ್ಗ. ಪ್ರವೃತ್ತಿಧರ್ಮವನ್ನು ಅನುಸರಿಸುವವನು, ಗೃಹಸ್ಥಧರ್ಮವನ್ನು ಚೆನ್ನಾಗಿಯೇ ಅನುಸರಿಸುವನು. ದೇವ ಪಿತೃಗಳನ್ನು ತೃಪ್ತಿಪಡಿಸಲು ನಾನಾವ್ರತಗಳನ್ನು ನಡೆಸುವನು. ಅವುಗಳಿಂದ ಪುಣ್ಯವನ್ನು ಗಳಿಸಿಕೊಳ್ಳುವನು. ಈ ಪುಣ್ಯಫಲವೇ, ಅವನು ಮರಣಿಸಿದ ಬಳಿಕ, ಅವನನ್ನು “ಧೂಮಾದಿಗತಿ’ಯಲ್ಲಿ ಎಂದರೆ, ಧೂಮ, ರಾತ್ರಿ, ಕೃಷ್ಣಪಕ್ಷ, ದಕ್ಷಿಣಾಯನ, ಪಿತೃಲೋಕ, ಆಕಾಶ, ಚಂದ್ರ - ಈ ಕ್ರಮವನ್ನು ಅನುಸರಿಸಿ, ಚಂದ್ರಲೋಕವನ್ನು ಸೇರುತ್ತಾನೆ. ಅಲ್ಲಿ ತನ್ನ ಕರ್ಮಫಲವನ್ನು ಅನುಭವಿಸುತ್ತಾ, ಅದು ಮುಗಿದ ಬಳಿಕ, ಅಲ್ಲಿಂದ ಹಿಂತಿರುಗಿ, ಆಕಾಶ, ವಾಯು, ಧೂಮ, ಮೇಘ ಎಂಬೀ ಕ್ರಮದಲ್ಲಿ, ಭೂಮಿಗೆ ಹಿಂತಿರುಗುವನು. ಚಂದ್ರಲೋಕವನ್ನು ಸೇರಿದವರು ಶಾಶ್ವತವಾಗಿ ಅಲ್ಲಿಯೇ ನೆಲಸಿರಲು ಸಾಧ್ಯವಿಲ್ಲವಾದ್ದರಿಂದ, ಅವರು ಭೂಮಿಗೆ ಬರಲೇಬೇಕಾಗುವುದು. ಇದರಿಂದ ಶಾಶ್ವತಸುಖವೆಂಬುದು ದೊರೆಯದು. 



ಪ್ರವೃತ್ತಿ ಧರ್ಮಕ್ಕೆ ಬದಲಾಗಿ ನಿವೃತ್ತಿ ಧರ್ಮವನ್ನು ಹಿಡಿಯುವ ಗೃಹಸ್ಥನು ತಾನು ಮಾಡುವ ಎಲ್ಲ ಕರ್ಮಗಳನ್ನೂ ಫಲಾಪೇಕ್ಷೆಯಿಲ್ಲದಂತೆ ಭಾಗವದರ್ಪಣ ರೂಪವಾಗಿ ನಡೆಸುವನು. ಅವನು ದೃಢಮನಸ್ಸುಳ್ಳವನಾಗಿ, ಇಂದ್ರಿಯಗಳನ್ನು ಜಯಿಸಿ,ಅರ್ಥ, ಕಾಮಗಳನ್ನು ಪ್ರಧಾನವಾಗಿ ಮಾಡಿಕೊಳ್ಳದೆ, ಸತ್ಸಂಗದಲ್ಲಿದ್ದುಕೊಂಡು, ತಾನು ಮಾಡುವ ಒಂದೊಂದು ಕರ್ಮವನ್ನೂ ಭಗವದರ್ಪಣ ಮಾಡುತ್ತಾ, ಶಮದಮಾದಿ ಸಾತ್ವಿಕ ಗುಣಗಳಿಂದ ಕೂಡಿದ್ವನಾಗಿ, ತನ್ನ ಜೀವನವನ್ನು ಕಳೆದ ಬಳಿಕ ಅವನು ಅರ್ಚಿರಾದಿಗತಿಯಲ್ಲಿ – ಎಂದರೆ, ಅರ್ಚಿ\(ಅಗ್ನಿ\), ಅಹಸ್ಸು, ಶುಕ್ಲಪಕ್ಷ, ಉತ್ತರಾಯಣ, ಸಂವತ್ಸರ, ವಾಯು, ಸೂರ್ಯ, ಚಂದ್ರ, ವಿದ್ಯುತ್ತು, ವರುಣ, ಇಂದ್ರ, ಬ್ರಹ್ಮಲೋಕಗಳ ಮಾರ್ಗವಾಗಿ- ಪರಮಪುರುಷನನ್ನೇ ನೇರವಾಗಿ ಸೇರುವನು. ಶಾಶ್ವತಸುಖವನ್ನೂ ಅನುಭವಿಸುವನು. 



ಪ್ರವೃತ್ತಿ ಧರ್ಮದಿಂದ ಧೂಮದಿಗತಿಯನ್ನು ಹಿಡಿದವರು ಮತ್ತೆ ಭೂಮಿಗೆ ಹಿಂತಿರುಗಲೇ ಬೇಕಾಗುವುದು. ನಿವೃತ್ತಿಧರ್ಮದಿಂದ ಭಕ್ತಿಯೋಗವನ್ನು ಹಿಡಿದವರು \(ಕಾಮಪುರುಷಾರ್ಥವನ್ನು ಉತ್ತಮರೀತಿಯಲ್ಲಿ ಬಳಸಿಕೊಳ್ಳುವವರು\) ಅರ್ಚಿರಾದಿ ಮಾರ್ಗದಿಂದ ಸೂರ್ಯಮಂಡಲದ ಮೂಲಕ, ಪರಮಪುರುಷೋಪಾಸನೆಯ ಬಲದಿಂದ, ಮತ್ತೆ ಹಿಂತಿರುಗಿ ಬರದಂಥ, ಶಾಶ್ವತಾನಂದವನ್ನು ತರುವಂಥ, ಪರಮಪದವನ್ನೇ ಪಡೆದುಕೊಳ್ಳುವನು. 



ಆಳ್ವಾರರುಹೇಳುತ್ತಾಳೆ: ಭೂಲೋಕವಾಸಿಗಳು ಆಶಿಸುವ ಮೂರು ಪುರುಷಾರ್ಥಗಳ ಪೈಕಿ, ’ಕಾಮ’ ಪುರುಷಾರ್ಥವನ್ನು, ಎಂದರೆ, ಭಕ್ತಿಯೋಗವನ್ನು, ಅಭ್ಯಾಸಮಾಡಿ, ಅದರ ಪೂರ್ಣಪ್ರಯೋಜನವನ್ನು ಪಡೆದುಕೊಳ್ಳೂವವರಿಗೆ ಮಿಕ್ಕೆರಡು ಪುರುಷಾರ್ಥಗಳು, ಎಂದರೆ, ಧರ್ಮ, ಅರ್ಥಗಳು ತಾವಾಗಿ ಒದಗಿ ಬರುತ್ತವೆ. ನಾಲ್ಕನೆಯ ಪುರುಷಾರ್ಥವೆಂದು ಹೇಳುವುದು ತೊಡಕಿನದು. ಅದನ್ನು ಸಾಧಿಸಿಕೊಳ್ಳುವುದು ಕಷ್ಟಸಾಧ್ಯವೆನ್ನುತ್ತಾರೆ. ಸೇರಲು ಸುಲಭವಲ್ಲದ ಸೂರ್ಯಮಂಡಲವನ್ನು ಸೇರಿ ಅದನ್ನು ಭೇದಿಸಿಕೊಂಡು, ಅದರ ನಡುವೆ ನೆಲಸಿರುವ ಪರಮಾತ್ಮನನ್ನು ಸೇರುವುದು ಅದು ಎಂದು ವಿವರಿಸಲಾಗಿದೆ. ಹೀಗೆ ಯಾರಾದರೂ ಅದನ್ನುಸಾಧಿಸಿಕೊಂಡಿರಬಹುದೇ? ನಮ್ಮ ಹತ್ತಿರದಲ್ಲಿಯೇ ಸುಂದರವಾದ ಮೊಲ ಇದ್ದರೂ, ಅದನ್ನು ತೊರೆದು, ಕಪ್ಪುಕಾಗೆಯನ್ನು ಹುಡುಕಿಕೊಂಡು ಹೋಗುವುದು ತರವೇ? ಕಾಮಪುರುಷಾರ್ಥವನ್ನು ಬದಿಗೊತ್ತಿ, ಕೈಗೆ ಅಬ್ಬದ ಮೋಕ್ಷವನ್ನು ಅರಸುವುದೇ? “ಏರಾರ್ ಮುಯಲ್ ವಿಟ್ಟು ಕಾಕ್ಕೈಪ್ಪಿನ್ ಪೋವದೇ” – ಭೂಲೋಕದ ಕಾಮ \(ಭಕ್ತಿ\) ಮಾರ್ಗಕ್ಕೂ, ಎಲ್ಲೋ ಆಕಾಶದಲ್ಲಿದೆ ಎನ್ನುವ ’ಅರ್ಚಿರಾದ’ ಮೋಕ್ಷಮಾರ್ಗಕ್ಕೂ ತಾರತಮ್ಯವನ್ನು ತೋರಿಸುವುದಕ್ಕಾಗಿಯೂ, ಯಾವುದನ್ನು ಅನುಸರಿಸುವುದರಿಂದ ಸುಲಭವಾಗಿ ಒಳ್ಳೆಯದಾಗುವುದೆಂದೂ ವಿವರಿಸುವುದಕ್ಕಾಗಿ ಆಳ್ವಾರರು ಸುಂದರವಾದ ಸಾಮ್ಯವೊಂದನ್ನು ಹೇಳುತ್ತಿದ್ದಾರೆ. 



“ಮುಯಲ್” ಎಂದರೆ ಮೊಲ. ಅದು ನೆಲದ ಮೇಲೆ, ಬಿಲಗಳಲ್ಲಿ ಜೀವಿಸುತ್ತಾ, ಗೆಡ್ಡೆ ಗೆಣಸು ಮುಂತಾದವನ್ನು ತಿನ್ನುತ್ತಾ ಬದುಕುವ ಸುಂದರವಾದ ಪುಟ್ಟ ಪ್ರಾಣಿ. ಓಟದಲ್ಲಿ ಅದನ್ನು ಮೀರಿಸುವ ಪ್ರಾಣಿ ವಿರಳ. ಇಷ್ಟಾದರೂ, ಪ್ರಯತ್ನಪಟ್ಟು, ಮೊಲವನ್ನು ಹಿಡಿದು ಹಾಕಲು ಸಾಧ್ಯ. ಮತ್ತು ಸಾಕಲೂಬಹುದು. ’ಕಾಕ್ಕೈ’ ಎಂದರೆ ಕಾಗೆ. ಅದೊಂದು ಮಾಂಸಾಹಾರಿಯಾದ ಕ್ರೂರಹಕ್ಕಿ. ಕಪ್ಪನೆಯ ಅದಕ್ಕೆ ಸೌಂದರ್ಯವೂ ಇದೆಯೇ? ಅತ್ಯಲ್ಪವಾದ ಹತ್ತಿರದ ಚಲನವಲನಗಳಿಂದ ಅದು ಹಾರಿಹೋಗುವಂಥಾದ್ದು, ಅದನ್ನು ಹಿಡಿಯುವುದಂತೂ ಸುಲಭವೇ ಅಲ್ಲ.



ಇದರಂತೆಯೇ, ಭಕ್ತಿ ಮಾರ್ಗ ಸುಂದರವೂ ಸುಲಭವೂ ಆದದ್ದು. ಯಾವ ಬಗೆಯ ಅಪಾಯದಿಂದಲೂ ಕೂಡದೆ ಇರುವುದು. ಭೂಮಿಯ ಮೇಲೆಯೇ ನಾನಾತಿರುಪತಿಗಳಲ್ಲಿ ನೆಲಸಿರುವ \(ಪ್ರಕೃತಿ ರಮ್ಯವೂ, ಬೆಟ್ಟಗಳ ಶಿಖರದಲ್ಲಿಯೂ ನೆಲಸಿರುವ\) ದಿವ್ಯಸುಂದರನಾದ ಅರ್ಚಾವತಾರಿಯಾದ ಭಗವಂತನ ಸಾನ್ನಿಧ್ಯವನ್ನು ಹುಡುಕಿಕೊಂಡು ಹೋಗುತ್ತಾ, ಎಷ್ಟೇ ಕಷ್ಟಕರವಾದರೂ, ಅದನ್ನು ಸಾಧಿಸಿಕೊಂಡು, ಪಡೆದುಕೊಳ್ಳಬಹುದು. ವೇದಶಾಸ್ತ್ರಗಳಲ್ಲಿ ವಿವರಿಸಿರುವ ಅರ್ಚಿರಾದಿ ಮೋಕ್ಷಮಾರ್ಗ ಬಹಳ ಕಠಿಣವಾದದ್ದು. ಅದಕ್ಕಾಗಿ ಎಷ್ಟು ಶ್ರಮಿಸಿದರೂ ಪಡೆದುಕೊಳ್ಳುವುದು ಸುಲಭವಲ್ಲದ್ದು.

ಏರ್ ಆರ್ = ಸೌಂದರ್ಯತುಂಬಿದ, ಇಳ ಮುಲೈಯೀರ್ = ಯೌವನದ ಮೊಲೆಗಳುಳ್ಳವರೇ, ಎನ್ ತನಕ್ಕೂ = ನನಗೆ, ಉಟ್ರದು ತಾನ್ = ಸಂಭವಿಸಿದ್ದು ತಾನೇ, \(ಕೇಳಿರಿ\), ಕಾರ್ ಆರ್ = ಕಾರ್ಮುಗಿಲನ್ನು ಹೋಲುವ \(ಕಪ್ಪುಬಣ್ಣ ತುಂಬಿದ\), ಕುೞಲ್ = ತಲೆಗೂದಲನ್ನು, ಎಡುತ್ತು ಕಟ್ಟಿ = ಎತ್ತಿಕಟ್ಟಿ, ಕದಿರ್ ಮುಲೈಯೈ = ಉಬ್ಬಿ ಕಾಣಿಸುವ ಮೊಲೆಯನ್ನು, ವಾರ್ ಆರ ವೀಕ್ಕಿ = ಕುಪ್ಪಸ\(ಬಲವಾಗಿ\) ಹೊಂದಿಕೊಂಡಿರುವಂತೆ ಬಿಗಿಸಿ, ಮಣಿ ಮೇಕಲೈ = ರತ್ನಗಳಿಂದ ಕೂಡಿದ ಒಡ್ಯಾಣವನ್ನು, ತಿರುತ್ತಿ = ಧರಿಸಿಕೊಂಡು \(ಅಲಂಕರಿಸಿಕೊಂಡು\) ಆರ್ ಆರ್ = ಸೊಬಗುತುಂಬಿದ, ಅಯಿಲ್ ವೇಲ್ ಕಣ್ = ವೇಲಾಯುಧದಂತೆ ತೀಕ್ಷ್ಣವಾದ ಕಣ್ಣುಗಳಿಗೆ, ಅಣ್ಜನತ್ತಿನ್ ನೀೞ್ ಅಣಿನ್ದು = ಕಾಡಿಗೆಯ ರಸವನ್ನು \(ಸಾರವನ್ನು\) ಬಳಿದು \(ಅಲಂಕರಿಸಿಕೊಂಡು\), ಶೀರ್ಕ್ ಆರ್ = ಶ್ರೇಶ್ಠತೆಯಿಂದ ಕೂಡಿದ, ಶೆೞು = ಸುಂದರವಾದ, ಪನ್ದು ಕೊಣ್ಡು = ಚೆಂಡನ್ನು ಎತ್ತಿಕೊಂಡು, ಆಡಿಯಾನಿನ್ ಱೇನ್ ನಾನ್ = ಆಡುತ್ತಾ ನಾನು ನಿಂತಿದ್ದೆ, ನೀರ್ ಆರ್ = ನೀರನ್ನು ಹೊಂದಿಕೊಂಡಿರುವ, ಕಮಲಮ್ ಪೋಲ್ = ಕಮಲದ ಹಾಗೆ, ಶೆಮ್ಕಣ್ = ಸುಮ್ದರಆದ ಕಣ್ಣುಗಳ, ಮಾಲ್ = ವ್ಯಾಮೋಹಕಾರ \(ಸರ್ವೇಶ್ವರ\) ಎನ್ ಱ ಒರುವನ್ = ಎಂಬ ಪುರುಷನೊಬ್ಬನು, ಪಾರೋರ್ ಹಳೆಲ್ಲಾಮ್ = ಭೂಲೋಕದವರೆಲ್ಲರೂ, ಮಹಿೞ = ಆನಂದಿಸುವಂತೆ, ಪಱೈ ಕಱಙ್ಗ = ಹರೆಯನ್ನು \(ಹರೆ ಎಂಬ ವಾದ್ಯವನ್ನು\) ನುಡಿಸಲು, ಶೀರ್ ಆರ್ =ಹೊಳೆಹೊಳೆಯುವ, ಕುಡಮ್ ಇರಣ್ಡು = ಎರಡು ಬಿಂದಿಗೆಗಳನ್ನು, ಏನ್ದಿ = ಧರಿಸಿ \(ಎತ್ತಿಕೊಂಡು\), ಶೆೞು = ವಿಸ್ತಾರವೂ ಸುಂದರವೂ ಆದ, ತೆರುವೇ = ಬೀದಿಯಲ್ಲಿ, ಆರ್ ಆರ್ ಎನ = ಯಾರುಯಾರು ಎಂದು, ಶೊಲ್ಲಿ = ಹೇಳುತ್ತಾ, ಆಡುಮ್ = ಆಡುವ \(ಕುಣಿದಾಡುವ\), ಅದು ಕಣ್ಡು = ಅದನ್ನು ಕಂಡವರಾಗಿ, ಏರ್ ಆರ್ = ಸುಂದರವಾದ, \(ಸೊಬಗು ತುಂಬಿದ\), ಇಳ ಮುಲೈಯಾರ್ ಅನ್ನೈಯರುಮ್ = ಯೌವನದ \(ಎಳೆಯ\) ಮೊಲೆಯ ತಾಯಂದಿರು, ಎಲ್ಲಾರುಮ್= ಎಲ್ಲರೂ, ವಾರಾಯೋ ಎನ್ ಱಾರ್ ಕ್ಕೂ = ’ನೀನೂ ಬರಲಾರೆಯಾ?’ ಎಂದವರಿಗೆ, ಶೆನ್ ಱೇನ್ = \(ಅವರೊಡನೆ\) ನಾನೂ ಕೂಡಿಕೊಂಡೆ \(ನಾನೂ ಹೋದೆ\), ಎನ್ ವಲ್ವಿನೈಯಾಲ್ = ನನ್ನ ಕ್ರೂರಪಾಪಕರ್ಮಗಳಿಂದ, ಕಾರ್ ಆರ್ = ಕಪ್ಪು ಹೊಂದಿದ \(ಕಪ್ಪು ಕೂಡಿದ\)ಮಣಿ ನಿಱಮುಮ್ = ಇಂದ್ರನೀಲಮಣಿಯ ಬಣ್ಣವನ್ನೂ, ಕೈವಳೈಯುಮ್ = ಕೈಬಳೆಗಳನ್ನೂ, ಕಾಣೇನ್ ನಾನ್ = ನಾನು ಕಳೆದುಕೊಂಡೆ, ಆರಾನುಮ್ = ಯಾರಾದರೂ, ಶೊಲ್ಲಿಟ್ರುಮ್ = ಹೇಳಿದರೂ \(ಹಿತನುಡಿದರೂ\), ಕೊಳ್ಳೇನ್ = ಅದನ್ನು ಮನಸ್ಸಿಗೆ ತಂದುಕೊಳ್ಳಲಾರೆನು. 



ಸೊಬಗು ತುಂಬಿದ ಎಳೆಯ \(ಯೌವನದ\) ಮೊಲೆಗಳವರೇ, ನನಗೆ ಸಂಭವಿಸಿದ್ದನ್ನು ಕೇಳಿರಿ: ಕಾರ್ಮುಗಿಲನ್ನು ಹೋಲುವ \(ಕಪ್ಪುಬಣ್ಣದ\) ತಲೆಗೂದಲನ್ನು ಎತ್ತಿಕಟ್ಟಿ, ಉಬ್ಬಿ ಕಾಣಿಸುವ ಮೊಲೆಗಳನ್ನು ಅವುಗಳ ಕುಪ್ಪಸ ಭದ್ರವಾಗಿ ಹೊಂದಿಕೊಂಡಿರುವಂತೆ ಬಿಗಿಸಿ, ರತ್ನಗಳಿಂದ ಕೂಡಿದ ಒಡ್ಯಾಣವನ್ನು \(ನಡುವಿಗೆ\) ತೊಟ್ಟು \(ಅಲಂಕರಿಸಿಕೊಂಡು\), ಸೊಬಗು ತುಂಬಿದ ವೇಲಾಯುಧದಂತೆ ಹೊಳೆಯುವ ಕಣ್ಣುಗಳಿಗೆ ಕಾಡಿಗೆಯ ಸಾರವನ್ನು ಬಳಿದು \(ಶೃಂಗರಿಸಿಕೊಂಡು\), ಸುಂದರವಾದ ಚೆಂಡನ್ನು ಎತ್ತಿಕೊಂಡು ಆಡುತ್ತಾ ನಾನು ನಿಂತಿದ್ದೆ. ನೀರನ್ನು ಹೊಂದಿಕೊಂಡಿರುವ ಕಮಲದ ಹಾಗೆ ಸುಂದರವಾದ ಕಣ್ಣುಗಳ ’ವ್ಯಾಮೋಹಕಾರ’ \(ಮನ್ಮಥ-ಸರ್ವೇಶ್ವರ\) ಎಂಬ ಪುರುಷನೊಬ್ಬನು, ಭೂಲೋಕದವರೆಲ್ಲರೂ ಆನಂದಿಸುವಂತೆ ಹರೆಯನ್ನು ನುಡಿಸಲು, ಹೊಳೆಹೊಳೆಯುವ ಎರಡು ಬಿಂದಿಗೆಗಳನ್ನು ಎತ್ತಿಕೊಂಡು, ಸುಂದರವೂ ವಿಸ್ತಾರವೂ ಆದ ಬೀದಿಯಲ್ಲಿಯಾರು ಯಾರು’ ಎಂದು ಹೇಳುತ್ತಾ ಕುಣಿದಾದುವುದನ್ನು ಕಂಡು, ಸೊಬಗು ತುಂಬಿದ ಎಳೆ ಮೊಲಯ ತಾಯಂದಿರು ಎಲ್ಲರೂ ’ನೀನೂ ಬಾರೆಯಾ?’ ಎಂದ ಅವರೊಡನೆ ನಾನೂ ಕೂಡಿ ಹೋದೆ. ಕ್ರೂರವಾದ ನನ್ನ ಪಾಪಕರ್ಮಗಳಿಂದ, ಕಪ್ಪು ತುಂಬಿದ ಇಂದ್ರನೀಲಮಣಿಯ ಬಣ್ಣವನ್ನೂ, ನನ್ನ ಕೈ ಬಳೆಗಳನ್ನೂ ನಾನು ಕಳೆದುಕೊಂಡೆನು. ಯಾರು ಎನು ಹೇಳಿದರೂ \(ಎಷ್ಟೆ ಹಿತನುಡಿದರೂ\) ನಾನು ನಂಬಲಾರೆನು. 



ಇಲ್ಲಿ ಆಳ್ವಾರರು ತಮ್ಮ ನಿಜಸ್ವರೂಪವನ್ನು ಮರಮಾಡಿಕೊಂಡಿದ್ದಾರೆ. ಚೆಲುವೆಯಾದ ಯುವತಿಯಾಗಿದ್ದಾರೆ. ಪರಮಪುರುಷನಾದ ಸರ್ವೇಶ್ವರನಲ್ಲಿ ವ್ಯಾಮೋಹಗೊಂಡಿದ್ದಾರೆ. ಅವರ ವಿರಹದ ಯಾತನೆಯನ್ನು ತಮ್ಮ ವಯಸ್ಸಿನ ಗೆಳತಿಯರೊಡನೆ ಹೇಳಿಕೊಳ್ಳುತ್ತಾರೆ. 



ಈಗ, ಪಾಶುರದ ನಾಯಕಿಯನ್ನು ಗಮನಿಸೋಣ. ಅವಳು ಸುಂದರಿ. ಯುವತಿ ಸುಸಂಸ್ಕೃತಳು. ತನ್ನ ರೂಪಕ್ಕೆ ತಕ್ಕಂತೆ ಅಲಂಕಾರ. ಕೈಯಲ್ಲಿ ಒಳ್ಳೆಯ ಹೂಚೆಂದು. ಅದನ್ನಾಡುತ್ತಾ ಏನೂ ಅರಿಯದ ಮುಗ್ದೆಯಂತೆ, ತನ್ನ ಮನೆಯ ಮುಂದಣ ಅಂಗಳದಲ್ಲಿದ್ದಾಳೆ. ಮನೆಯ ಮುಂದೆ ದೊಡ್ಡದೊಂದು ವಿಸ್ತಾರವಾದ ಸುಂದರವಾದ ಬೀದಿ. ಅದರಲ್ಲಿ ದಿವ್ಯಮನೋಹರಮೂರ್ತಿಯೊಬ್ಬನು ಕೈಯಲ್ಲಿ ಹರೆಯನ್ನು ಬಾರಿಸುತ್ತಾ ಬರುತ್ತಿದ್ದಾನೆ. ಅವನ ತಲೆಯ ಮೇಲೆ ಹೊಳೆಹೊಳೆಯುವ ಎರಡು ಬಿಂದಿಗೆಗಳಿವೆ. ಹರೆಯನ್ನು ಬಾರಿಸುತ್ತಾ, ನಡುನಡುವೆ, ಅವನು ಕೂಗಿ ಹೇಳುತ್ತಾನೆ. “ಯಾರು ಯಾರಿಗೆ ನನ್ನೊಡನೆ ಕೂಡಿ ಕುಣಿದಾಡಲು ಇಷ್ಟವಿದೆ? ಎಲ್ಲರೂ ಬನ್ನಿ” ಬೀದಿಯಎಲ್ಲ ಯುವತಿಯರೂ, ತಾಯಂದಿರೂ, ಎಂದರೆ, ಎಲ್ಲ ಸುಂದರಯುವತಿಯರೂ ಅವನೊಡನೆ ಕೂಡಿ ಕುಣಿದಾಡಲು ಬೀದಿಗೆ ಸಡಗರದಿಂದ ಬಂದರು. ನಾಯಕಿಯನ್ನು ಕಂಡರು. ಅವಳು ತನ್ನ ಕಪ್ಪುತಲೆಗೂದಲನ್ನು ಬಾಚಿ, ಸುಂದರವಾದ ತುರುಬನ್ನಾಗಿ ಕಟ್ಟಿದ್ದಳು. ಅವಳ ವಿಶಾಲವಾದ ಹೊಳೆಯುವ ಕಣ್ಣಿಗೆ ಕಾಡಿಗೆ ತಿದ್ದಿದ್ದಳು. ಎದೆಯಲ್ಲಿ ಉಬ್ಬಿ ಕಾಣುತ್ತಿದ್ದ ಸುಂದರವಾದ ಮೊಲೆಗಳನ್ನು ಬಿಗಿಯಾಗಿ ಕುಪ್ಪಸದಿಂದ ಬಿಗಿದಿದ್ದಳು. ನಡುವಿಗೆ ರತ್ನಮಯವಾದ ಒಡ್ಯಾಣವನ್ನು ತೊಟ್ಟಿದ್ದಳು. ಹೀಗೆ ಎಲ್ಲ ಬಗೆಯಲ್ಲೂ ಅಣಿಯಾಗಿದ್ದ ನಾಯಕಿಯನ್ನು ಕಂಡ ಅವರು “ಆ ಪರಮ ಸುಂದರನನ್ನು ನೋಡು. ಆನು ಕೊಡದ ಕುಣಿತವನ್ನಾಡುತ್ತಾನೆ. ಅವನ ತಲೆಯ ಮೇಲೆ ಎರಡು ಹೊಳೆಯುವ ಬಿಂದಿಗೆಗಳಿವೆ\! ನಾವೆಲ್ಲರೂಅವನ ಕರೆಗೆ ಓಗೊಟ್ಟು ಹೋಗುತ್ತಿದ್ದೇವೆ. ನೀನೂ ನಮ್ಮೊಡನೆ ಬಾರೆಯಾ? ನಾಯಕಿಯೂ ಭ್ರಮಿಸಿದಳು. ಅವರೊಡನೆ ಹೋಗಿ ಆ ಪರಮಪುರುಷನೊಡನೆ ಬೆರೆತು ಆನಂದಿಸಬೇಕೆಂಬ ಆಶೆಯಿಂದ ಅವಳು ಹೊರಟಳು. ಆದರ ಅವಳಿಗೆ ಸಂಭವಿಸಿದ್ದೇನೆ ಏಕೆ? 



ನಾಯಕಿ ತನ್ನ ಅನುಭವವನ್ನು ತನ್ನ ಗೆಳತಿಯರಿಗೆ ಹೇಳುತ್ತಾಳೆ: ನೋಡಿ, ಗೆಳತಿಯರೇ\! ನನ್ನ ಅನುಭವ ಹೇಗಿದೆ? ನನ್ನ ಬಣ್ಣವೇಹೋಯಿತು. ನಾನೀಗ ಬಿಳಿಚಿಕೊಂಡಿದ್ದೇನೆ. ನನ್ನಕೈಬಳೆಗಳೇ ಕಾಣದಾದವು. ಅವನ್ನು ಕೊಂಡೊಯ್ದವರಾರು? ಕಾಣೆನಲ್ಲ\! ಆ ದಿವ್ಯಸುಂದರನೊಡನೆ ಮೈಮರೆತು ಆನಂದಿಸಬೇಕೆಂದು ನಾನಿಲ್ಲಿಗೆ ಬಂದೆ. ನನ್ನ ಕ್ರೂರವಾದ ಪಾಪಕರ್ಮಗಳೇ ನನ್ನ ಈ ದುಃಸ್ಥಿತಿಗೆ ಕಾರ್‍ಅಣವಿರಬೇಕು. ಯಾರು ಈ ವಿಷಯದಲ್ಲಿ ನನಗೆ ಎಷ್ಟು ಬಗೆಯಾಗಿ ಹಿತನುಡಿಯಬಂದರೂ ನಾನದನ್ನು ನಂಬಲಾರೆ.



ಪರಮಾತ್ಮನನ್ನು ಅಗಲಿರುವುದು ವಿರಹ. ಅದಕ್ಕಾಗಿಯೇ ಕೊರಗು, ಸಂಕಟ, ಇತ್ಯಾದಿಗಳು. ಅಗಲಿದವನನ್ನು ಹೇಗಾದರೂ ಕೂಡಿಕೊಳ್ಳಲೇಬೇಕು. ಆಗಲೇ ನಿತ್ಯಾನಂದ ಸುಖ\!

ಅಱಿವು ಅೞಿನ್ದು = ಬುದ್ಧಿಕೆಟ್ಟು, ತೀರಾ = ಮುಗಿಯದ \(ತೀರದ\), ಉಡಮ್ಬೊಡು = ದೇಹದಿಂದ ಕೂಡಿ, ಪೇದು = ಹುಚ್ಚಿಯಂತೆ, ಉಱುವೇನ್ = ಇರುವವಳನ್ನು, ಕಣ್ಡು =ನೋಡಿ, ಇರಙ್ಗಿ = ಕನಿಕರಿಸಿ, ಏರ್ ಆರ್ = ಸೊಬಗು ತುಂಬಿದ, \(ಸುಂದರಿಯಾದ\), ಕಿಳಿಕಿಳವಿ = ಗಿಳಿಯ\(ಶಾಸ್ತ್ರದ\) ಮುದುಕಿಯಾದ, ಎಮ್ ಅನೈ ತಾನ್ = ನಮ್ಮ ತಾಯಿಯೇ, ವನ್ದು = ಬಂದು, ಎನ್ನೈ = ನನ್ನನ್ನು, ಶೀರ್ ಆರ್ = ಪವಿತ್ರವಾದ, ಶೆೞು = ಸುಂದರವಾದ, ಪುೞುದಿ = ಶ್ರೀಪಾದ ಧೂಳಿಯನ್ನು, ಕಾಪ್ಪು ಇಟ್ಟು = ರಕ್ಷೆಯನ್ನು ಹಚ್ಚಿ, ಶೆಮ್ ಕುಱಿಂಜಿ = ಕೆಂಪುಗುಲಗಂಜಿಯ, ತಾರ್ ಆರ್ ನಱುಮಾಲೈ = ಮಣಿಗಳಿಂದ ಕೂಡಿದ ಸುಂದರವಾದ ಹಾರವನ್ನು, ಶಾತ್ತಱ್ ಕು = ಧರಿಸಿದವನಿಗೆ, ತಾನ್ = ತಾನು, ಪಿನ್ನುಮ್ = ಹಿಂದೆ ಎಂದೂ, \(ಮತ್ತೆ\), ನೇರಾದನ = ಮಾಡದ, ಒನ್ ಱು = ಒಂದನ್ನು, ನೇರ್ ನ್ದಾಳ್ = ಮಾಡಿದಳು, ಅದನಾಲುಮ್ = ಅದರಿಂದಲೂ, ತೀರಾದು = ತೀರದು, ಎನ್ಶಿನ್ದೈನೋಯ್ = ನನ್ನ ಮನಸ್ಸಿನ ಸಂಕಟ, ತೀರಾದು ಎನ್ ಪೇದುಱವು = ನನ್ನ ಹುಚ್ಚು ಸಹತೀರದು, ವಾರಾದು ಮಾಮೈ = ದೇಹಕಾಂತಿಯು ಬರದು, ಅದು ಕಣ್ಡು = ಅದನ್ನು ಕಂಡು, ಮಟ್ರು ಆಙ್ಗೇ = ಮತ್ತೆಅಲ್ಲಿಯೇ, ಆರಾನುಮ್ = ಯಾರಾದರೂ, ಮೂದು = ಮುದುಕಿಯರಾಗಿ, ಅಱಿಯುಮ್ = ತಿಳಿವಳಿಕೆಯುಳ್ಳವರಾದ, ಅಮ್ಮನೈಮಾರ್ = ತಯಂದಿರು ಶೊಲ್ಲುವಾರ = ಹೇಳತೊಡಗಿದರು, ಪಾರೋರ್ = ಭೂವಾಸಿಗಳು, ಶೊಲಪ್ಪಡುಮ್ = ಹೇಳುವಂತೆ, ಕಟ್ಟುಪ್ಪಡುತ್ತಿಲ್ ಏಲ್ = ಬಲವಂತಪಡಿಸಿದರೆ, ಆರಾನುಮ್ = ಯಾರಾದರೂ, ಮೆಯ್ ಪಡುವನ್ = ಸತ್ಯವನ್ನು ಹೇಳುವನು \(ಹೊರಪಡಿಸುವನು\), ಎನ್ ಱಾರ್ = ಎನ್ನುತ್ತಾರೆ. 



ಹೀಗೆ ಬುದ್ಧಿಗೆಟ್ಟು, ಮುಗಿಯದ \(ತೀರದ\) ದೇಹದಿಂದ ಕೂಡಿ, ಹುಚ್ಚಿಯಂತೆ ಇರುವ ನನ್ನನ್ನು ಕಂಡು, ಸುಂದರಿಯಾದ ಗಿಳಿಯ \(ಶಾಸ್ತ್ರದ\) ಮುದುಕಿಯಾಗಿ ನಮ್ಮ ತಾಯಿಯೇ ಬಂದು, ನನಗೆ ಪವಿತ್ರವೂ ಸುಂದರವೂ ಆದ ಶ್ರೀಪಾದಧೂಳಿಯನ್ನು ರಕ್ಷೆಯಾಗಿ ಇಟ್ಟು, ಕೆಂಪು ಗುಲಗಂಜಿಯ ಮಣಿಗಳಿಂದ ಕೂಡಿದ ಸುಂದರವಾದ ಹಾರವನ್ನು ಧರಿಸಿದವನ ಬಳಿಯಲ್ಲಿ \(ಕರೆದೊಯ್ದು\) ತಾನು ಹಿಂದೆ ಎಂದೂ ಮಾಡದೆ ಇದ್ದ ಒಂದನ್ನು ಮಾಡಿದಳು. ಅದರಿಂದಲೂ ನನ್ನ ಮನಸ್ಸಿನ ಸಂಕಟ ತೀರಲಿಲ್ಲ. ನನ್ನ ಹುಚ್ಚು ಸಹ ತೀರಲಿಲ್ಲ. ನನ್ನ ದೇಹಕಾಂತಿ ಬರಲಿಲ್ಲ. ಅದನ್ನು ಕಂಡು, ಮತ್ತೆ ಅಲ್ಲಿಯೇ ಯಾರಾದರೂ ತಿಳಿವಳಿಕೆಯುಳ್ಳ ವೃದ್ಧತಾಯಂದಿರು ಹೇಳತೊಡಗಿದರು: “ಭೂಲೋಕವಾಸಿಗಳು ಹೇಳುವಂತೆ ಬಲವಂತಪಡಿಸಿದರೆ, ಯಾರಾದರೂ ನಿಜವನ್ನು ಹೊರಹಾಕುವಳು” ಎಂದು. 



ಇಲ್ಲಿ ನಾಯಕಿಯು ತನ್ನ ವಿರಹವೇದನೆಯನ್ನೂ, ತನ್ನ ತಾಯಿಯೂ ಇತರ ತಿಳಿದವರೂ ತನಗಾಗಿ ನಡೆಸಿದ ಚಿಕಿತ್ಸೆಯನ್ನು ಹೇಳಿಕೊಳ್ಳುತ್ತಾಳೆ.



ನಾಯಕಿ ಹೇಳುತ್ತಾಳೆ: ನನ್ನಮೈ ಬಡವಾಯಿತು. ಮೈಬಣ್ಣ ಹೋಯಿತು. ಕೈಬಳೆಗಳು ಕಳಚಿಬಿದ್ದವು. ಬುದ್ಧಿಕೆಟ್ಟಿತು. ದೇಹ ಮಾತ್ರ ತೀರಿಕೊಳ್ಳಲಿಲ್ಲ. ನಾನು ಹುಚ್ಚಿಯಂತಾದೆ. ನಾನು ಹೀಗೆ ಅನುಭವಿಸುವುದನ್ನು ಕಂಡ ನಮ್ಮ ತಾಯಿ ತಾನೇ ಗಿಳಿಶಾಸ್ತ್ರ ಹೇಳುವ ಮುದುಕಿಯಂತೆ ವೇಷಧರಿಸಿ, ನನ್ನ ಬಳಿಗೆ ಬಂದಳು. ನನ್ನ ಹಣೆಗೆ ಶ್ರೀಪಾದಧೂಳಿಯನ್ನು ರಕ್ಷೆಯಾಗಿ ಇರಿಸಿದಳು. ಅನಂತರ ನನ್ನನ್ನು ಮಂತ್ರವಾದಿಯ ಬಳಿಗೆ ಕರೆದೊಯ್ದಳು. ಅವನು ಕೆಂಪುಗುಲಗಂಜಿಯ ಮಣಿಗಳ ಹಾರಗಳನ್ನು ಧರಿಸಿದ್ದನು. ಎಂದೂ ಮಾಡದ ಮಂತ್ರಚಿಕಿತ್ಸೆಯನ್ನು ಅವನಿಂದ ಮಾಡಿಸಿದಳು. ಅದರಿಂದ ನನ್ನ ಹುಚ್ಚು ತೀರಲಿಲ್ಲ. ನನ್ನ ಮೈತುಂಬಿಕೊಳ್ಳಲಿಲ್ಲ. ಮೈಬಣ್ಣ ಮತ್ತೆ ಬರಲಿಲ್ಲ. ಅಲ್ಲಿಯೇ ಹತ್ತಿರದಲ್ಲಿದ್ದ ಜ್ಞಾನವೃದ್ಧೆಯರು ಹೇಳಿದರು. “ಎಲ್ಲಾ ಜನರೂ ಮಾಡುವಂತೆ ಮಾಡಿದರೆ, ಅವಳು ನಿಜವನ್ನು ಹೇಳುತ್ತಾಳೆ: 



ಜನಸಾಮಾನ್ಯರು ಮಾಡುವುದಾದರೂ ಏನು? ರೋಗಿಯ ಮನೋವೈಕಲ್ಯಕ್ಕೆ ಕಾರಣವೇನೆಂದು ಮೃದುವಾದ ರೀತಿಯಲ್ಲಿ ಕಂಡುಕೊಳ್ಳಹೋಗುವುದಿಲ್ಲ. ಅದಕ್ಕೆ ಬದಲಾಗಿ, ರೋಗಿಯನ್ನು ಶಿಕ್ಷಿಸಿ, ಹಿಂಸಿಸಿ, ಬಲವಂತಪಡಿಸುತ್ತಾರೆ. ಅದರಿಂದ ಅವಳು \(ರೋಗಿಯು\) ತನ್ನಹಿಂಸೆಯನ್ನು ತಡೆಯಲಾರದೆ ನಿಜವನ್ನು ಹೇಳಿಬಿಡುತ್ತಾಳೆ – ಎಂದು ಅವರ ನಂಬಿಕೆ. ಇದು ವಾಸ್ತವವೇ?

ಸಾಮಾನ್ಯವಾಗಿ, ಮಂತ್ರವಾದಿ ಅಥವಾ ಕಣಿಗಾತಿಯಾಗಿ ಆವೇಶಗೊಂಡಿರುವುದಕ್ಕೆ ಕೆಲವು ಲಕ್ಷಣಗಳನ್ನು ಅವರು ತೋರ್ಪಡಿಸುತ್ತಾರೆ. ಅವರ ಕಣ್ಣುಗಳು ಕೆಂಡದಂತೆ ಕಿಡಿಕಿಡಿಯಾಗುತ್ತಾನೆ. ಮೈನಡುಗುತ್ತದೆ, ಮೈನೀರೇಳುತ್ತದೆ. ಬೆವರು ಸುರಿಯುತ್ತದೆ. ಅವರ ಮಾತೂ ಸಂಜ್ಞೆಗಳು ವಿಚಿತ್ರವಾಗಿರುತ್ತವೆ. ಕೆಲವು ವೇಳೆ, ಉನ್ಮತ್ತಳಾದ ವ್ಯಕ್ತಿಯನ್ನು ಬೈದು, ಹೊಡೆದು, ಹಿಂಸಿಸಿ, ಅವಳ ಮನಸ್ಸಿನಲ್ಲಿರುವುದನ್ನು ಹೊರಕೆಡಹುವುದಕ್ಕೆ ಯತ್ನಿಸುವುದೂ ಉಂಟು. 



ಆದರೆ, ಈ ಪಾಶುರದ ಕಣಿಗಾತಿ ಬೇರೆ ಯಾರೂ ಅಲ್ಲ. ತನ್ನ ತಾಯಿಯೇ ಅವಳಾದ್ದರಿಂದ, ತನ್ನ ಮಗಳನ್ನು ಹಿಂಸಿಸುವುದಿಲ್ಲ. ದಂಡಿಸುವುದಿಲ್ಲ. ಆವೇಶಗೊಂಡವಳಂತೆ ನಟಿಸಿದರೂ, ಸಹ, ಶಾಂತರೀತಿಯಲ್ಲಿ ವಿಷಯವನ್ನು ಹೇಗೋ ಹೊರಕೆಡಹುತ್ತಾಳೆ. ಮಗಳ ಮನಸ್ಸಿನಲ್ಲಿರುವುದೇನು ಎಂಬುದನ್ನು ಅವಳು ಅರಿತಿದ್ದಾಳೆ. ಅದನ್ನೇ ಸಾಮಾನ್ಯಕಣಿಗಾತಿಯ ಹಾಗೆ, ಅರೆಕೊರೆ ಮಾತಿನಿಂದಲೂ, ಅಭಿನಯಗಳಿಂದಲೂ ಹೊರಪಡಿಸುತ್ತಾಳೆ. ನೆರೆದ ಯುವತಿಯರಿಗೆ ಅವಳು ಹೇಳುತ್ತಾಳೆ.

ವೇರ್ ಆರ್ = ಬೆವರು ತುಂಬಿದ, ನುದಲ್ = ಹಣೆಯ, ಮಡವಾಳ್ = ಶ್ರೀಮತಿಯು, ವೇಱು ಓರ್ ಕಲತ್ತು ಇಟ್ಟು = ಬೇರೊಂದು ಪಾತ್ರೆಯಲ್ಲಿಟ್ಟು, ನಾರ್ ಆರ್ = ನಾರಿನಿಂದಾದ, ಉಱಿ ಏಟ್ರಿ = ನೆಲುವಿನ ಮೇಲಿಟ್ಟು \(ನೆಲವನ್ನ ಏರಿಸಿ\) ನನ್ ಹು ಅಮೈಯ ವೈತ್ತು = ಚೆನ್ನಾಗಿ ಕೂಡಿತ್ತು, ಪೋರ್ ಆರ್ = ಯುದ್ಧಕ್ಕೆ ಸಿದ್ಧವಾದ, ವೇಲ್ = ವೇಲಾಯುಧದಂತೆ \(ಹೊಳೆಯುವ\), ಕಣ್ = ಕಣ್ಣುಗಳ, ಮಡವಾಳ್ = ಶ್ರೀಮತಿಯು, ಪೊಯ್ ಉಱಕ್ಕಮ್ = ಕಪಟ ನಿದ್ದೆಯನ್ನು, ಓರಾದವನ್ ಪೋಲ್ = ಏನೂ ಅರಿಯದವನಂತೆ, ಉಱಙ್ಗಿ = ನಿದ್ರಿಸಿ, ಅಱವುಉಟ್ರು = ಚೆನ್ನಾಗಿ ತಿಳಿದೆದ್ದು, ತಾರ್ ಆರ್ = ಹಾರಗಳಿಂದ ತುಂಬಿದ, ತಡತೋಳ್ ಹಳ್ = ದೀರ್ಘವಾದ ತೋಳುಗಳು, ಉಳ್ಳವುಮ್ = ಸಾಧ್ಯವಾದಷ್ಟೂ, ಕೈನೀಟ್ಟಿ ಕೈಚಾಚಿ, ಆರಾಧ = ಸಮೃದ್ಧಿಯಾದ, ವೆಣ್ನೆಯ್ ವಿೞುಙ್ಗು = ಬೆಣ್ಣೆಯನ್ನು ನುಂಗಿ, ಅರುಹು ಇರುನ್ದ = ಮಗ್ಗುಲಲ್ಲಿದ್ದ, ಮೋರಾರ್ = ಮಜ್ಜಿಗೆ ತುಂಬಿದ್ದ, ಕುಡಮ್ = ಕೊಡವನ್ನು, ಉರುಟ್ಟಿ = ಉರುಡಿಸಿ, ಮುನ್ = ಮೊದಲಲ್ಲಿ, ಕಿಡನ್ದ = \(ತಾನು\) ಮಲಗಿದ್ದ ತಾನತ್ತೀ = ಸ್ಥಳದಲ್ಲಿಯೇ, ಓರಾದವನ್ ಪೋಲ್ = ಸ್ಥಳದಲ್ಲಿಯೇ, ಓರಾದವನ್ ಪೋಲ್ = ಏನೂ ಅರಿಯದವನಂತೆ, ಕಿಡನ್ದಾನೈ = ಮಲಗಿದ್ದವನನ್ನು, ಕಣ್ಡ ಅವಳುಮ್ = ನೋಡಿದ ಅವಳೂ ವಾರಾ = ಅಲ್ಲಿಗೆ ಬರಲು, ತಾನ್ ವೈತ್ತದು = ತಾನು ಇಟ್ಟದ್ದನ್ನು \(ಇಟ್ಟ ಹಾಗೆ\) ಕಾಣಾಲ್ = ಕಾಣಳು, ವಯಿರ್ = ಹೊಟ್ಟೆಯನ್ನು, ಅಡಿತ್ತು = ಹೊಡೆದುಕೊಂಡು, ಇಙ್ಗು = ಇಲ್ಲಿಗೆ, ಆರಾರ್ = ಯಾರು ಯಾರು, ಪುಹುದು ವಾರ್ = ನುಗ್ಗಿ \(ನುಸುಳಿ\) ಬರುವರು, ಐಯರ್ ಇವರ ಅಲ್ಲಾಲ್ = ಈ ಸ್ವಾಮಿಯವರಲ್ಲದೆ, ನೀರ್ ಆಮ್ ಇದು ಶೆಯ್ದೀರ್ = ಇದನ್ನು ಮಾಡಿದವರು ನೀವೇ ಸರಿ, ಎನ್ ಱು = ಎಂದು, ಓರ್ ನೆಡು ಕಯಿಟ್ರಾಲ್ = ಒಂದು ಉದ್ದನಾದ ಹಗ್ಗದಿಂದ, ಊರಾರ್ ಹಳ್ ಎಲ್ಲಾರುಮ್ ಕಾಣ = ಊರಿನವರೆಲ್ಲರೂ ಕಾಣುವಂತೆ, ಉರಲೋಡೇ = ಒರಳಿಗೆ, ತೀರಾ = ತೀರದ, ವೆಹುಳಿಯಳ್ ಆಯ್ = ಕೋಪಗೊಂಡವಳಾಗಿ, ಚಿಕ್ಕನ = ಬಲವಾಗಿ, ಆರ್ ತ್ತು = ಕಟ್ಟಿ, ಅಡಿಪ್ಪ = ಹೊಡೆಯಲು, ಆರಾ = ಹೆಚ್ಚಾದ, ವಯಿಟ್ರ್‍ಇನೋಡು = ಹೊಟ್ಟೆಯುರಿಯಿಂದ, ಆಟ್ರಾದಾನ್ =ಸಂಕಟಪಟ್ಟವನು.



ಯಾರಿಂದ ಈ ಭೂಮಿಯು ಅಳೆಯಲ್ಪಟ್ಟಿತೋ, ಯಾರಿಂದ ಲಂಕೆಯು ಪುಡಿಪುಡಿಯಾಗುವಂತೆ ಬಿದ್ದು ನಾಶವಾಯಿತೋ, ಯಾರಿಂದ ಕಲ್ಲಿನ ಮಳೆಯಿಂದ \(ಜನ ದನಗಳು\) ಕಾಪಾದಲ್ಪಟ್ಟಿತೋ, ಯಾರಿಂದ ಕಡಲಿನ ನೀರು ಕಡೆಯಲ್ಪಟ್ಟಿತೋ, ಅವನೇ ಕಾಣಿರಿ ಊರಿನ ದನಕರುಗಳ ಮಂದೆಯನ್ನು ಮೇಯಿಸಿದನು. ಅವನೇ ಲೋಕಗಳೆಲ್ಲವನ್ನೂ ಉಂಡು ಉಗುಳಿದನು; ಅವನೇ ತೃಪ್ತಿ ತಾರದ ಸ್ವಭಾವದವನಾಗಿರುವನು. ಒಂದು ಕಾಲದಲ್ಲಿ ಅಲ್ಲಿ \(ನಂದಗೋಕುಲದಲ್ಲಿ\) ಒಂದು ಸಲ ಸುಂದರವಾದ ಸೀರೆಯನ್ನುಟ್ಟ ನಿತಂಬಗಳ, ಸುಂದರವಾದ ಕಾಲುಗಳ, ಚೆಂದುಟಿಗಳ, ಬಿಗಿದ ಕುಪ್ಪಸದಿಂದ ಶೋಭಿಸುವ ಸುಂದರವಾದ ಮೊಲೆಗಳುಳ್ಳವಳು ಕಡಗೋಲನ್ನು ಭದ್ರವಾಗಿ ಹಿಡಿದು, ಸುಂದರವಾದ ನಡುವು ನೋಯುವಂತೆ ಎಷ್ಟೋಕಾಲದವರೆಗೆ ಸೊಗಸಾದ ಮೊಸರನ್ನು ಕಡೆದು, ತೇಲಿಬಂದ ಬೆಣ್ಣೆಯನ್ನು ಬೆವರು ತುಂಬಿದ ಹಣೆಯ ಶೀಮತಿಯು ಬೇರೊಂದು ಪಾತ್ರೆಯಲ್ಲಿಟ್ಟು, ನಾರಿನಿಂದಾದ ನೆಲುವನ್ನು ಏರಿಸಿ, ಚೆನ್ನಾಗಿ ಕೂಡಿಟ್ಟು, ಯುದ್ಧಕ್ಕೆ ಸಿದ್ಧವಾದ ವೇಲಾಯುಧದಂತೆ ಹೊಳೆಯುವ ಕಣ್ಣುಗಳ ಶ್ರೀಮತಿಯು ನಿದ್ರಿಸಿದಾಗ, ಏನೂ ಅರಿಯದವನಂತೆ, ಸುಳ್ಳುನಿದ್ದೆಯನ್ನು ನಟಿಸಿದವನು ಚೆನ್ನಾಗಿ ತಿಳಿದೆದ್ದು, ಹಾರಗಳಿಂದ ಶೋಭಿಸುವ ಉದ್ದನಾದ ತೋಳುಗಳನ್ನು ಸಾಧ್ಯವಾದಷ್ಟೂ ಕೈಚಾಚಿ, ಸಮೃದ್ಧಿಯಾದ ಬೆಣ್ಣೆಯನ್ನು ನುಂಗಿ, ಮಗ್ಗುಲಲ್ಲಿದ್ದ ಮಜ್ಜಿಗೆ ತುಂಬಿದ ಕೊಡವನ್ನು ಉರುಡಿಸಿ, ಮೊದಲು ತಾನು ಮಲಗಿದ್ದ ಸ್ಥಳದಲ್ಲಿಯೇ ಏನೂ ಅರಿಯದವನಂತೆ ಮಲಗಿದ್ದವನನ್ನು ಕಂಡ ಅವಳು ಅಲ್ಲಿಗೆ ಬರಲು, ತಾನಿಟ್ಟ ಹಾಗೆ ಯಾವುದನ್ನೂ ಕಾಣದೆ ಹೊಟ್ಟೆಯನ್ನು ಹೊಡೆದುಕೊಂಡು ಅಲ್ಲಿಗೆ ಬೇರೆ ಯಾರೂ ಬರಲು ಸಾಧ್ಯವಿಲ್ಲವಾದ್ದರಿಂದ, ಈ ಕೆಲಸ ಮಾಡಿರುವವರು ನೀವೇ ಸರಿ’ ಎಂದು ಒಂದು ಉದ್ದನಾದ ಹಗ್ಗದಿಂದ ಊರಿನವೆಲ್ಲರೂ ಕಾಣುವ ಬಗೆ, ಬಹಳ ಕೋಪದಿಂದ, ಒರಳಿಗೆ ಬಲವಾಗಿ ಬಿಗಿದು ಹೊಡೆಯಲು, ಬಹಳ ಸಂಕಟಪಟ್ಟು ಅತ್ತವನು ಅವನು.



ಈ ಪಾಶುರ ಭಾಗದಲ್ಲಿ, ಭಗವಂತನ ನಾನಾ ಅವತಾರಗಳ ವೈಖರಿಯನ್ನು ವಿವರಿಸಲಾಗಿದೆ. ಭೂಮಿಯನ್ನು ಅಳೆದದ್ದು ತ್ರಿವಿಕ್ರಮನಾಗಿ. ಲಂಕೆಯನ್ನು ನಾಶಮಾಡಿದ್ದು ಶ್ರೀರಾಮನಾಗಿ. ದೇವೇಂದ್ರನ ಕಲ್ಲುಮಳೆಯಿಂದ ಗೋವಳರನ್ನೂ ಗೋವುಗಳನ್ನೂ ರಕ್ಷಿಸಿದ್ದೂ, ನಂದಗೋಕುಲದಲ್ಲಿ ಯಶೋದೆ ಕೂಡಿಟ್ಟ ಬೆಣ್ಣೆಯನ್ನು ಕಪಟರೀತಿಯಲ್ಲಿ ತಿಂದು, ಮಜ್ಜಿಗೆಯನ್ನು ಕುಡಿಸಿ, ಮಾಡಿದ ಚೇಷ್ಟೆಯನ್ನು ಸಹಿಸದೆ ಯಶೋದೆಯಿಂದ ಒರಳಿಗೆ ಬಿಗಿಸಿಕೊಂಡು, ಹೊಡೆತ ತಿಂದು ಅತ್ತಿದ್ದು ಶ್ರೀಕೃಷ್ಣನಾಗಿ, ಮತ್ತು ಪಾಲ್ಗಡಲನ್ನು ಕಡೆಯಿಸಿ, ಅಮೃತವನ್ನು ಪಡೆದು, ಅದನ್ನು ದೇವತೆಗಳಿಗೆ ಹಂಚಿದ್ದು ಶ್ರೀಮನ್ನಾರಾಯಣನಾಗಿಯೇ. 



ಕಣಿಗಾತಿಯು ಇತರ ಹೆಣ್ಣುಮಕ್ಕಳಿಗೆ ಇವುಗಳನ್ನೆಲ್ಲಾ ಅವರಿಗೆ ತಿಳಿಯುವಂಥ ಮಾತುಗಳಿಂದಲೇ ವಿವರಿಸಿ ಹೇಳಿದ್ದಾಳೆ.

ಅನ್ ಱಿಯುಮ್ = ಅಲ್ಲದೆ, ನೀರ್ ಆರ್ = ನೀರು ತುಂಬಿರುವ, ನೆಡು = ಆಳವಾದ, ಕಯತ್ತೈ = ಮಡುವನ್ನು, ಶೆನ್ ಱು = ಹೋಗಿ, ಅಲೈಕ್ಕ = ಚೆನ್ನಾಗಿ ಕಲಕಿ, ನಿನ್ ಱು = ಅದರಲ್ಲಿದ್ದುಕೊಂಡು, ಉರಪ್ಪಿ = ಘರ್ಜಿಸಿ, ಓರ್ ಆಯಿರಮ್ ಪಣಮ್ = ಸಾವಿರ ಹೆಡೆಗಳ, ವೆಙ್ಗೋ ಇಯಲ್ = ನನ್ನೊಡನೆ \(ಹೋರಾಡಲು\), ವಾರಾಯ್ ಎನ್ ಱು = ಮಟ್ರು = ಮತ್ತು, ಅದನ್ = ಅದರ ಮತ್ತ ಹತ್ತು = ತಲೆಯಮೇಲೆ ಬಾ ಎಂದು ಶೀರ್ ಆರ್ =\(ತನ್ನ\) ಸುಂದರವಾದ \(ಪವಿತ್ರವಾದ\), ತಿರುವಡಿಯಾಲ್ = ತಿರುವಡಿಗಳಿಂದ, ಪಾಯ್ ನ್ದಾನ್ = ನರ್ತನಮಾಡಿದನು. 



ಅಲ್ಲದೆ, ನೀರು ತುಂಬಿರುವ ಆಳವಾದ ಮಡುವನ್ನು ಹೊಕ್ಕು, ಅದನ್ನು ಚೆನ್ನಾಗಿ ಕಲಕಿ, ಅಲ್ಲಿಯೇ ಇದ್ದುಕೊಂಡು, ಘರ್ಜಿಸಿ, ಒಂದು ಸಾವಿರ ಹೆಡೆಗಳ ಯಮಸ್ವಭಾವದ ನಾಗನನ್ನು ತನ್ನೊಡನೆ ಹೋರಾಟಕ್ಕೆ ಬಾಯೆಂದು \(ಘರ್ಜಿಸಿ\) ಅದರ ತಲೆಯ ಮೇಲೆ ತನ್ನ ಪವಿತ್ರವಾದ ತಿರುವಡಿಯಿಂದ ನರ್ತನ ಮಾಡಿದನು. 



ಇಲ್ಲಿ ಕೃಷ್ಣಾವತಾರದ ಕಾಳೀಯ ಮಥನದ ಒಂದು ಪ್ರಸಂಗವನ್ನು ಹೇಳುತ್ತಾರೆ. ಕಾಳಿಂದಿ ಮಡುವಿನಲ್ಲಿ ಬಹುಕಾಲದಿಂದ ನೆಲಸಿದ್ದ ಕಾಳೀಯನೆಂಬ ವಿಷಸರ್ಪವನ್ನು ಅಲ್ಲಿಂದ ಅಟ್ಟಿ, ಮಡುವಿನ ನೀರನ್ನು ದನಕರುಗಳು ಕುಡಿಯಲು ಯೋಗ್ಯವಾಗಿರುವಂತೆ ಮಾಡಿದ ಪ್ರಸಂಗವಿದು.

ಈರ ವಿಡುತ್ತವಟ್ಕು ಮೂತ್ತೋನೈ, ವೆನ್ನರಹಮ್



೪೧. ಶೇರಾ ವಹೈಯೇ ಶಿಲೈಕುಳಿತ್ತಾನ್, ಶೆನ್ದುವರ್ ವಾಯ್

ವಾರಾರ್ ವನಮುಲೈಯಾಳ್ ವೈದೇವಿ ಕಾರಣಮ್,



೪೨. ಏರಾರ್ ತಡನ್ದೋಳಿರಾವಣನೈ, ಈರೈನ್ದು

ಶೀರಾರ್ ಶಿರಮಱುತ್ತು ಶೆಟ್ರುಹನ್ದ ಶೆಙ್ಗಣ್ ಮಾಲ್- 



ತನ್ ಶೀದೈಕ್ಕು = ಅವನ ಸೀತೆಗೆ, ನೇರ್ ಆವನ್ = ಸಮನಾಗುತ್ತೇನೆ,ಎನ್ ಱು = ಎಂದು, ಓರ್ ನಿಶಾಚರಿ ತಾನ್ = ಒಬ್ಬ ನಿಶಾಚರಿಯೇ, ವನ್ದಾಳೈ = ಬಂದವಳನ್ನು, ಕೂರ್ ಆರ್ ನ್ದ = ಬಲುಹರಿತವಾದ \(ಹರಿತವನ್ನು ಹೊಂದಿದ\), ವಾಳಾಲ್ = ಕತ್ತಿಯಿಂದ, ಕೊಡಿಮೂಕ್ಕುಮ್ = ತುದಿಮೂಗನ್ನೂ, ಕಾದು ಇರಣ್ಡುಮ್ = ಎರಡು ಕಿವಿಗಳನ್ನೂ, ಈರಾ ವಿಡುತ್ತು = ಕೊಯ್ದು ಓಡಿಸಿ, ಅವಟ್ಕು = ಅವಳ, ಮೂತ್ತೋನೈ = ಅಣ್ಣನನ್ನು, ವೆಮ್ ನರಹಮ್ = ಕ್ರೂರವಾದ ನರಕವನ್ನು, ಶೇರಾ ವಹೈಯೇ = ಶೇರದ ರೀತಿಯಲ್ಲೇ, ಶಿಲೈ = ಬಿಲ್ಲನ್ನು, ಕುಳಿತ್ತಾನ್ = ಬಗ್ಗಿಸಿ \(ಅವನನ್ನು ನಾಶಪಡಿಸಿದನು\), ಶೆನ್ದುವರ್ ವಾಯ್ = ಚೆಂದುಟಿಯ, ವಾರ್ ಆರ್ = ಕುಪ್ಪಸ ಬಿಗಿದ, ವನ್ ಮುಲೈಯಾಳ್ = ಸುಂದರವಾದ ಮೊಲೆಯವಳಾದ, ವೈದೇವಿ = ವೈದೇಹಿಯ, ಕಾರಣಮ್ = ಕಾರಣವಾಗಿ, ಏರ್ ಆರ್ = ಸುಂದರವಾದ \(ಸೊಬಗುತುಂಬಿದ\) ತಡತೊಳಿ = ಉದ್ದನಾದ ಶೊಳಿನವನಾಡ, ಇರಾವಣನೈ = ರಾಅಣನ, ಈರ್ ಐನ್ದು = ಹತ್ತು, ಶಿರಮ್ = ತಲೆಗಳನ್ನು, ಅಱುತ್ತು = ಕತ್ತರಿಸಿ, ಶೆಟ್ರು= \(ಅವನನ್ನು\) ಕೊಂದು, ಉಹನ್ದ = ಹರ್ಷಿಸಿದ, ಶೆಙ್ಗಣ್ ಮಾಲ್ = ಸುಂದರವಾದ ಕಣ್ಣುಗಳುಳ್ಳ ವ್ಯಾಮೋಹಕಾರಿಯು. 



ಅವನ ಸೀತೆಗೆ ನಾನು ಸಮನಾಗಿದ್ದೇನೆ ಎಂಬು ಒಬ್ಬ ನಿಶಾಚರಿಯೇ ಬಂದವಳನ್ನು ಬಲು ಹರಿತವಾದ ಕತ್ತಿಯಿಂದ ಅವಳ ತುದಿಮೂಗನ್ನೂ ಎರಡು ಕಿವಿಗಳನ್ನೂ ಕೊಯ್ದು ಓಡಿಸಿ, ಅವಳ ಅಣ್ಣನನ್ನು ಕ್ರೂರವಾದ ನರಕವನ್ನು ಶೇರದ ಹಾಗೆ ಬಿಲ್ಲನ್ನು ಬಗ್ಗಿಸಿದನು \(ಅವನನ್ನು ನಾಶಪಡಿಸಿದನು\). ಚೆಂದುಟಿಯವಳೂ, ಕುಪ್ಪಸ ಬಿಗಿದ ಸುಂದರವಾದ ಮೊಲೆಗಳುಳ್ಳವಳೂ ಆದ ವೈದೇಹಿಯ ಕಾರಣವಾಗಿ ಸೊಬಗುತುಂಬಿದ ಉದ್ದವಾದ ತೋಳಿನವನಾದ ರಾವಣನ ಹತ್ತುತಲೆಗಳನ್ನು ಕತ್ತರಿಸಿ, ಅವನನ್ನು ಕೊಂದು ಹರ್ಷಿಸಿದ ಸುಂದರವಾದ ಕಣ್ಣುಗಳ ವ್ಯಾಮೋಹಕಾರಿಯು. 



ಈ ಪಾಶುರಭಾಗದಲ್ಲಿ ಎರಡು ಪ್ರಮೇಯಗಳು ಬಂದಿವೆ. ಮೊದಲನೆಯದು ಶೂರ್ಪನಖಿಯ ಗರ್ವಭಂಗ; ತಾನೇ ಸೀತೆಗಿಂತ ಮೇಲೆಂದು ರಾಮನಲ್ಲಿ ಲಲ್ಲೆಯಾಡುತ್ತಾ ಬಂದ ರಾಕ್ಷಸಿಯ ಕಿವಿಮೂಗುಗಳನ್ನು ಕೊಯ್ದು, ಅವಳ ಅಣ್ಣನಾದ ರಾವಣನ ಬಳಿಗೆ ಕಳುಹಿಸಿದ್ದು. ಎರಡನೆಯದು, ಸೀತೆಯ ನಿಮಿತ್ತವಾಗಿ ಆದ ರಾವಣಾಸುರನ ವಧೆ: ಅಪಮಾನಗೊಂಡ ಶೂರ್ಪನಖಿ ರಾವಣನಲ್ಲಿಗೆ ಹೋಗಿ, ರಾಮನ ಹೆಂಡತಿಯೂ ಅತಿಸುಂದರಿಯೂ ಆದ ಸೀತೆಯನ್ನು ಕದ್ದುತರುವಂತೆ ಪ್ರಚೋದಿಸಿದಳು. ಅವನು ಹಾಗೆಯೇ ಮಾಡಿದ. ಮೃತ್ಯುವನ್ನು ತನ್ನ ಮನೆಯಲ್ಲಿ ಸೇರಿಸಿಕೊಂಡ. ಆ ಸೀತೆಯನ್ನು ಬಂಧನದಿಂದ ಬಿಡಿಸುವ ಕಾರಣದಿಂದ, ಶ್ರೀರಾಮನು ಲಂಕೆಗೆ ಹೋಗಿ, ತನ್ನ ಕೋದಂಡದ ಸಹಾಯದಿಂದಲೇ ರಾವಣನ ಹತ್ತು ತಲೆಗಳನ್ನು ಕತ್ತರಿಸಿ, ಅವನನ್ನು ಕೊಂದು ಹಾಕಿದನು. ಅಂಥ ಪರಮಸುಂದರನಾದ ವ್ಯಾಮೋಹಕನು \(ಇವಳ\) ನಾಯಕ?

ಪೋರ್ ಆರ್ = ಯುದ್ಧವನ್ನು ಹೊಂದಿಕೊಂಡಿರುವ, ನೆಡುವೇಲೋನ್= ಉದ್ದವಾದ ವೇಲಾಯುಧವನ್ನುಳ್ಳ, ಪೊನ್ ಪೆಯರೋನ್ = ಹಿರಣ್ಯಕಶಿಪುವಿನ, ಆಹತ್ತೈ = ದೇಹವನ್ನು, ಕೂರ್ ಆರ್ ನ್ದ = ಬಲುಹರಿತವಾದ \(ಹರಿತ ತುಂಬಿದ\), ವಳುಹಿರಾ = ಬಲವಾದ ಉಗುರುಗಳಿಂದ, ಕೀಣ್ಡು = ಸೀಳಿ, ಕುಡಲ್ ಮಾಲೈ= ಕರುಳಿನ ಹಾರವನ್ನು, ಶೀರ್ ಆರ್ = ಸೊಬಗು ತುಂಬಿದ, ತಿರುಮಾರ್ ಬಿನ್ ಮೇಲ್ = ಪವಿತ್ರವಾದ ಎದೆಯ ಮೇಲೆ, ಕಟ್ತಿ = ಧರಿಸಿ, ಶೆಙ್ಗುರು = ಕೆಂಪು ರಕ್ತವು, ಶೋರಾ = ಸುರಿಯುವಂತೆ, ಕಿಡನ್ದಾನೈ = ಬಿದ್ದವನನ್ನು, ಕುಙ್ಕುಮಮ್ = ಕುಂಕುಮದಂತೆ ಕೆಂಪಾದ, ತೋಳ್ = ತೋಳುಗಳನ್ನು, ಕೊಟ್ಟಿ = ಪುಡಿಪುಡಿಮಾಡಿ, ಆರಾ = ಘರ್ಜಿಸುತ್ತಾ, ಎೞುನ್ದಾನ್ = ಉದ್ಭವಿಸಿದವನು, ಅರಿ ಉರು ಆಯ್ = ನರಹರಿ ರೂಪಿಯಾಗಿ. 



ಯುದ್ಧವನ್ನು ಹೊಂದಿಕೊಂಡಿರುವ, ಉದ್ದನಾಗಿರುವ ವೇಲಾಯುಧವನ್ನುಳ್ಳ ಹಿರಣ್ಯಕಶಿಪುವಿನ ದೇಹವನ್ನು, ಹರಿತ ತುಂಬಿದ ಬಲವಾದ ಉಗುರುಗಳಿಂದ ಸೀಳಿ, ಕರುಳಿನ ಹಾರವನ್ನು ಸುಂದರವಾದ ಪವಿತ್ರವಾದ ತನ್ನ ಎದೆಯಮೇಲೆ ಧರಿಸಿ, ಕೆಂಪುರಕ್ತವು ಸುರಿಸುತ್ತಾ ಬಿದ್ದವನ ಕೆಂಪಾದ ತೋಳುಗಳನ್ನು ಪುಡಿಪುಡಿಮಾಡಿ, ಘರ್ಜಿಸುತ್ತಾ ಉದ್ಭವಿಸಿದವನು ನರಹರಿರೂಪಿಯಾಗಿ. 



ಇಲ್ಲಿ ಭಗವಂತನ ನರಸಿಂಹಾವತಾರವನ್ನು ವಿವರಿಸಲಾಗಿದೆ. ಹರಿದ್ವೇಷಿಯಾಗಿ, ಹರಿಯನ್ನು ಭಜಿಸುತ್ತಿದ್ದ ತನ್ನ ಮಗನೇ ಆದ ಪ್ರಹ್ಲಾದನನ್ನು ಚಿತ್ರಹಿಂಸೆಗೆ ಗುರಿಮಾಡಿದ, ಮಹಾಬಲಿಷ್ಠನಾದ ಹಿರಣ್ಯಕಶಿಪುವನ್ನು ಭಗವಂತನು ನರಹರಿರೂಪಿಯಾಗಿ ತನ್ನ ಕೈಉಗುರುಗಳಿಂದಲೇ ಸೀಳಿಕೊಂದು ಹಾಕಿದ ವಿಷಯ ಇದು.

ಪೇರ್ ವಾಮನಾಹಿಯ ಕಾಲತ್ತು, ಮೂವಡಿ ಮಣ್



೪೬. ತಾರಾ ಯೆನಕ್ಕೆನ್ ಱು ವೇಣ್ಡಿಚ್ಚಲತ್ತಿನಾಲ್

ನೀರೇಟ್ರುರಿಹೆಲ್ಲಾಮ್ ನಿನ್ ಱಳನ್ದಾನ್ ಮಾವಲಿಯೈ – 



ಅನ್ ಱಿಯುಮ್ = ಅಲ್ಲದೆ, ಪೇರ್ = ಹೆಸರು, ವಾಮನ್ = ವಾಮನ ಎಂದು, ಆಹಿಯ = ಆದ, ಕಾಲತ್ತು = ಕಾಲದಲ್ಲಿ, ಮೂವಡಿ ಮಣ್ = ಮೂರು ಹೆಜ್ಜೆಗಳ ನೆಲವನ್ನು, ತಾರಾ ಎನ್ ಱು= ಕೊಡು ಎಂದು, ವೇಣ್ಡಿ = ಬೇಡಿ, ನೀರ್ ಏಟ್ರು = ನೀರೆರಿಸಿಕೊಂಡು, ಚಲತ್ತಿನಾಲ್ = ಛಲದಿಂದ ಉಲಹೆಲ್ಲಾಮ್ = ಲೋಕಗಳೆಲ್ಲವನ್ನೂ, ನಿನ್ ಱು= ನೆಟ್ಟಗೆ ನಿಂತು, ಅಳನ್ದಾನ್ = ಅಳೆದುಕೊಂಡನು, ಮಾವಲಿಯೈ = ಬಲಿಚಕ್ರವರ್ತಿಯನ್ನು. 



ಅಲ್ಲದೆ, ’ವಾಮನ’ ಎಂಬ ಹೆಸರನ್ನಿಟ್ಟುಕೊಂಡ ಕಾಲದಲ್ಲಿ ಮೂರು ಹೆಜ್ಜೆಗಳ ನೆಲವನ್ನು ಕೊಡೆಂದು ಬಲಿಚಕ್ರವರ್ತಿಯನ್ನು ಯಾಚಿಸಿ, ಛಲದಿಂದ, ತ್ರಿವಿಕ್ರಮನಾಗಿ ಬೆಳೆದು ನಿಂತು, ಎಲ್ಲಾ ಲೋಕಗಳನ್ನೂ ಅಳೆದುಕೊಂಡು ಬಲಿಚಕ್ರವರ್ತಿಯನ್ನು ಗೆದ್ದನು.

೪೮. ಪೇರಾಮಲ್ ತಾಙ್ಗಿಕ್ಕಡೈನ್ದಾನ್,

ತಿರುತ್ತುೞಾಯ್ ತ್ತಾರಾರ್ ನ್ದ ಮಾರ್ ವನ್. 



ಆರಾದ =ಮುಗಿಯದ, ಪೋರಿಲ್ = ಯುದ್ಧದಲ್ಲಿ, ಅಶುರರ್ ಹಳುಮ್ = ಅಸುರರೂ, ತಾನುಮ್ ಆಯ್ = ತಾನೂ ಆಗಿ, ಕಾರ್ ಆರ್ = ಕಾರ್ಮುಗಿಲನ್ನು ಹೋಲುವ \(ಕಾರ್ಮುಗಿಲು ಹುಟ್ಟುವ\) ವರೈ = ಪರ್ವತವನ್ನು, ನಟ್ಟು = \(ಕಡಗೋಲಾಗಿ\) ನೆಟ್ಟು, ನಾಹಮ್ = ನಾಗ\(ಸರ್ಪವನ್ನು\)ನನ್ನು, ಕಯಿಱ್ ಆಹ = ಹಗ್ಗವಾಗಿ, ಪೇರಾಮಲ್ = ಸ್ಥಳ ಬದಲಾಗದಂತೆ \(ಮುಳುಗಿ ಹೋಗದಂತೆ\), ತಾನುಮ್ = ತಾನೂ ಕೂಡಿಕೊಂಡು, ಕಡೈನ್ದಾನ್ = ಕಡೆದನು, ತಿರುತುಱಾಯ್ ತಾರ್ ಆರ್ ನ್ದ = ಪವಿತ್ರವಾದ ತುಲಸಿಯ ಹಾರದಿಂದ ತುಂಬಿದ, ಮಾರ್ ವನ್ = ಎದೆಯುಳ್ಳವನು.



ಮುಗಿಯದ ಯುದ್ಧದಲ್ಲಿ ಅಸುರರೂ ತಾನೂ ಆಗಿ, ಕಾರ್ಮುಗಿಲಿನಂಥ ಪರ್ವತವನ್ನು ನೆಟ್ಟು, ನಾಗನನ್ನು ಹಗ್ಗವಾಗಿ \(ಸುತ್ತಿ\), ಸ್ಥಳ ಬದಲಾಗದಂತೆ \(ಮುಳುಗದಂತೆ\) ತಾನು ಕೂಡಿಕೊಂಡು, ಪವಿತ್ರವಾದ ತುಲಸಿಯಹಾರದಿಂದ ಶೋಭಿಸುವ ಎದೆಯುಳ್ಳವನು \(ಪಾಲ್ಗಡಲನ್ನು\) ಕಡೆದನು. 



ಇಲ್ಲಿ ಭಗವಂತನ ಕೂರ್ಮಾವತಾರದ ಪ್ರಸಂಗ ಬಂದಿದೆ. ದೇವಾಸುರರಿಗೆ ಮುಗಿಯದ ಯುದ್ಧವೊದಗಿತು. ಆಗ ಭಗವಂತನೇ ಅವರ ನಡುವೆ ಸಂಧಿ ಮಾಡಿದನು. ಅವರಿಬ್ಬರೂ ಕೂಡಿ ಪಾಲ್ಗಡಲನ್ನು ಕಡೆದರೆ, ಅದರಿಂದ ಅಮೃತವು ಉತ್ಪತ್ತಿಯಾಗುವುದೆಂದೂ ಅದನ್ನು ಕುಡಿದರೆ, ಅವರೆಲ್ಲರೂ ಅಮರರಾಗುವರೆಂದೂ ಹೇಳಿ ಒಪ್ಪಿಸಿದನು. ಮಂದರಪರ್ವತವನ್ನೇ ಕಡಗೋಲನ್ನಾಗಿಯೂ, ವಾಸುಕಿಯನ್ನೇ ಕಡೆಯುವ ಹಗ್ಗವನ್ನಾಗಿಯೂ ಮಾಡಿಕೊಂಡು ಕಡೆಯುವುದಕ್ಕೆಂದು ಮಂದರ ಪರ್ವತವನ್ನು ಕಡಲಲ್ಲಿಳಿಸಿದಾಗ ಅದು ಪೂರ್ತಿ ಮುಳುಗದ ತೇಲಿಕೊಂಡಿರುವಂತೆ ಅದರಡಿಯಲ್ಲಿ ಭಗವಂತನೇ ಮಹಾಕೂರ್ಮನಾಗಿ ರೂಪಗೊಂಡು ತನ್ನ ಬೆನ್ನ ಮೇಲೆ ಅದನ್ನು ಇರಿಸಿಕೊಂಡು, ಪಾಲ್ಗಡಲನ್ನು ದೇವಾಸುರರಿಂದ ಕಡೆಯಿಸಿ, ಅಮೃತವನ್ನು ಪಡೆದು ಅದನ್ನು ಮೋಹಿನಿರೂಪದಲ್ಲಿ ದೇವತೆಗಳಿಗೆ ಮಾತ್ರವೇ ಹಂಚಿದನು.

೫೦. ನೀರಾರ್ ಮಲರ್ ಕ್ಕಮಲಮ್ ಕೊಣ್ಡೋರ್ ನೆಡುಙ್ಗೈಯಾಲ್,

ನಾರಾಯಣ ಓ ಮಣಿವಣ್ಣಾ ನಾಹಣೈಯಾಯ್.

ತೀರಾದ ಶೀಟ್ರತ್ತಾಱ್ ಶೆನ್ ಱಿರಣ್ಡು ಕೂಱಾಹ,



೫೨. ಈರಾವದನೈಯಿಡರ್ ಕಡಿನ್ದಾ ನೆಮ್ಬೆರುಮಾನ್,

ಪೇರಾಯಿರ ಮುಡೈಯಾನ್, ಪೇಯ್ ಪೆಣ್ಡೀರ್ ನುಮ್ಮಹಳೈ, 



೫೩. ತಡ = ದೊಡ್ಡ, ಮಾಲ್ ವರೈ ಪೋಲುಮ್ = ಬೆಟ್ಟಾನ್ನು ಹೋಲುವ, ಪೋರ್ ಆನೈ = ಬಲಿಷ್ಠ ಸಲಗವು, ಪೊಯ್ ಹೈವಾಯ್ = ಮಡುವಿನಲ್ಲಿ, ಕೊಳ್ ಪಟ್ಟು = ಮೊಸಳೆಬಾಯಿಗೆ ಸಿಕ್ಕಿ, ನಿನ್ ಱು = ನಿಂತು, ಅಲಱಿ = ಗೋಳಾಡಿ \(ಅರಚಾಡಿ\), ನೀರ್ ಆರ್ = ನೀರು ತುಂಬಿದ, ಮಲರ್ = ಅರಳಿದ, ಕಮಲಮ್ = ಕಮಲದ ಹೂವನ್ನು, ಓರ್ ನೆಡುಕೈಯಾಲ್ ಕೊಣ್ಡು = ಒಂಟಿಯಾದ ಉದ್ದನಾದ ಕೈಯಲ್ಲಿ ಎತ್ತಿಕೊಂಡು, ನಾರಾಯಣಾ = ನಾರಾಯಣನೇ, ಓ ಮಣಿವಣ್ಣಾ = ಓ ನೀಲಮಣಿವಣ್ಣನೇ, ನಾಹ ಅಣೈಯಾಯ್ = ನಾಗನನ್ನು ಹಾಸುಗೆಯಾಗಿ ಉಳ್ಳವನೇ \(ಶೇಷಶಯನನೇ\), ವಾರಾಯ್ = ಬಾರಯ್ಯ, ಎನ್ ಆರ್ ಇಡರೈ = ನನ್ನ ತುಂಬು ಸಂಕಟವನ್ನು, ನೀಕ್ಕಾಯ್ = ನೀಗಿಸಯ್ಯಾ, ಎನ = ಎನ್ನಲು, ವೆಹುಣ್ಡು = ಕೋಪಗೊಂಡು, ತೀರಾದ = ತೀರದೆ ಇರುವಂಥ \(ಅತಿಶಯವಾದ\), ಶೀಟ್ರತ್ತಾಲ್ = ಕೋಪದಿಂದ, ಶೆನ್ ಱು = ಹೋಗಿ, ಇರಣ್ಡು ಕೂೞಾಹ = ಎರಡು ಭಾಗವಾಗಿ, ಈರಾ = ಸೀಳಿ, \(ಕತ್ತರಿಸಿ\), ಅದನೈ = ಅದರ ಇಡರ್ = ದುಃಖ\(ಸಂಕಟ\)ವನ್ನು, ಕಡಿನ್ದಾನ್ = ನಿವಾರಿಸಿದನು, ಎಮ್ಬೆರುಮಾನ್ = ಸರ್ವೇಶ್ವರನು, ಪೇರ್ ಆಯಿರಮ್ = ಸಾವಿರ ಹೆಸರುಗಳನ್ನು, ಉಡೈಯಾನ್ = ಉಳ್ಳವನು, ಪೇಯ್ ಪೆಣ್ಡೀರ್ = ಹುಚ್ಚು \(ಬುದ್ಧಿಯಿಲ್ಲದ\) ಹೆಂಗಸರೇ, ನುಮ್ = ನಿಮ್ಮ, ಮಹಳೈ = ಮಗಳನ್ನು, ತೀರಾ = ತೀರದಂಥ \(ಬಲುಹೆಚ್ಚಾದ\), ನೋಯ್ ಶೆಯ್ ದಾನ್ = ನೋವನ್ನುಂಟುಮಾಡಿದ್ದಾನೆ, ಎನ = ಎಂದು, ಉರೈತ್ತಾಳ್ = ಹೇಳಿದಳು. 



ಬಲುದೊಡ್ಡ ಬೆಟ್ಟವನ್ನು ಹೋಲುವ ಬಲಿಷ್ಠ ಸಲಗವು, ಮಡುವಿನಲ್ಲಿ ಮೊಸಳೆಯ ಬಾಯಿಗೆ ಸಿಕ್ಕಿ, ನಿಂತು, ಅರಚುತ್ತಾ ನೀರು ತುಂಬಿದ ಅರಳಿದ ಕಮಲದ ಹೂವನ್ನು ಒಂದು ಉದ್ದನಾದ ಕೈಯಲ್ಲಿ ಎತ್ತಿಕೊಂಡು “ನಾರಾಯಣಾ, ಓ ಮಣಿವಣ್ಣಾ, ಶೇಷಶಯನಾ, ಬಾರಯ್ಯ\! ನನ್ನ ತುಂಬು ಸಂಕಟವನ್ನು ನೀಗಿಸಯ್ಯ” ಎನ್ನಲು, ಅಡಗಿಸಲಾಗದ ಕೋಪದಿಂದ, ಹೋಗಿ, ಎರಡು ಭಾಗವಾಗಿ ಸೀಳಿ, ಅದರ ಸಂಕಟವನ್ನು ಸರ್ವೇಶ್ವರನು ನಿವಾರಿಸಿದನು: ಬುದ್ಧಿಯಿಲ್ಲದ ಹೆಂಗಸರೇ ಆ ಸಾವಿರ ನಾಮಗಳನ್ನುಳ್ಳವನು ನಿಮ್ಮ ಮಗಳಿಗೆ ಬಲು ಹೆಚ್ಚಾದ \(ತೀರದ\) ನೋವನ್ನುಂಟುಮಾಡಿದ್ದಾನೆ ಎಂದು ಹೇಳಿದಳು. 



ಇದುವರೆಗೆ ಭಗವಂತನ ಗುಣಗಾನಮಾಡಿದ ಕಣಿಗಾತಿಯು ಮತ್ತೊಂದು ನಿದರ್ಶನವನ್ನು ಕೊಟ್ಟು, ತನ್ನ ಮಾತನ್ನು ಮುಗಿಸಿದಳು: ಇದು ಗಜೇಂದ್ರಮೋಕ್ಷದ ವಿಷಯ. ಗತ್ಯಂತರವಿಲ್ಲದೆ, ಮಡುವಿನಲ್ಲಿ ಮೊಸಳೆಯ ಬಾಯಿಗೆ ಸಿಕ್ಕಿಬಿದ್ದ ಸಲಗನು ಭಗವಂತನಲ್ಲಿ ಮೊರೆಯಿಡಲು, ಅದರ ಸಂಕಟವನ್ನೂ ಕೂಡಲೇ ಸ್ವಾಮಿಯು ನಿವಾರಿಸಿದನು. ದುಷ್ಟರನ್ನು ಶಿಕ್ಷಿಸುವ ದುಷ್ಟರನ್ನು ರಕ್ಷಿಸುವ ಸರ್ವೇಶ್ವರನೇ ಈಗ ನಿಮ್ಮ ಮಗಳಿಗೆ ತನ್ನ ವಿಷಯದಲ್ಲಿ ಅತಿಶಯವಾದ ಪ್ರೇಮವನ್ನುಂಟುಮಾಡಿ, ತೀರದ ನೋವನ್ನು ಕೊಡುತ್ತಿದ್ದಾನೆ. ಹೆಂಗಳೇ, ನೀವು ಬುದ್ಧಿಯಿಲ್ಲದಂತೆ ಅವಳ ವಿಷಯದಲ್ಲಿ ನಡೆದುಕೊಳ್ಳಬಾರದು ಎಂದಳು.

ನಾನ್ = ನಾನು, ಅವನೈ = ಅವನ ಕಾರ್ ಆರ್ = ಕಾರ್ಮುಗಿಲನ್ನು ಹೋಲುವ, ತಿರುಮೇನಿ = ದಿವ್ಯರೂಪವನ್ನು ಕಣ್ಡದುವೇ = ಕಂಡದ್ದೇ, ಕಾರಣಮ್ = ಕಾರಣವಾಗಿ, ಪೇರಾ = ಪ್ರಸಿದ್ಧವಾಗಿ, \(ಹೆಸರುಗಳನ್ನು\) ಪಿದಟ್ರಾ = ಒದರುತ್ತಾ, ತಿರಿತರುವನ್ = ಅಲೆದಾಡುತ್ತಿದ್ದೇನೆ, ಪಿನ್ನೈಯುಮ್ = ಮತ್ತೆ \(ಅದೂ ಅಲ್ಲದೆ\), ಈರಾ = ಮೃದುವಾಗಿ, ಪುಹುದಲುಮ್ = ಪ್ರವೇಶಿಸುತ್ತಾ, ಇ ಉಡಲೈ = ಈ ದೇಹವನ್ನು, ತಣ್ ವಾಡೈ = ತಂಗಾಳಿಯು, ಶೋರಾ = ಕೃಶಗೊಳಿಸುತ್ತಾ, ಅಱುಕ್ಕುಮ್ = ಕೊಯ್ಯುತ್ತಿರುವ, ವಹೈ = ಬಗೆಯನ್ನು \(ರೀತಿಯನ್ನು\), ಅೞಿಯೇನ್ = ಅರಿಯೆನು, ಶೂೞ್ ಕುೞಲಾರ್ = ತುಂಬಿದ ತಲೆಗೂದಲಿನವರೇ, ಆರಾನುವರು, ಎನ್ನುಮ್ = ಎಂಬ ಅದರ, ಪೞಿಯೈ = ಅಪಮಾನವು, ವಾರಾಮಲ್ = ಬರದಂತೆ, ಕಾಪ್ಪದಱ್ ಕು = ರಕ್ಷಿಸುವುದಕ್ಕೆ \(ರಕ್ಷಿಸಿಕೊಳ್ಳುವುದಕ್ಕೆ\), ಆಗದೆ ವಾೞ್ ವು = ಬಾಳ್ವೆಯನ್ನು, ಇರುನ್ದು = ಇದ್ದುಕೊಂಡು, ಒೞಿನ್ದೇನ್ = ಕಳೆಯುತ್ತಿದ್ದೇನೆ.



ನಾನು ಕಾರ್ಮುಗಿಲನ್ನು ಹೋಲುವ ಅವನ ದಿವ್ಯರೂಪವನ್ನು ಕಂಡದ್ದೇ ಕಾರಣವಾಗಿ, ಅವನ ಪ್ರಸಿದ್ಧವಾದ ಹೆಸರುಗಳನ್ನು ಒದರುತ್ತಾ ಅಲೆದಾಡುತ್ತಿದ್ದೇನೆ. ಅದೂ ಅಲ್ಲದೆ, ತಂಗಾಳಿಯು ಮೃದುವಾಗಿ ಈ ದೇಹವನ್ನು ಪ್ರವೇಶಿಸುತ್ತಾ, ನನ್ನನ್ನು ಕೃಶಗೊಳಿಸುತ್ತಾ, ಕೊಯ್ಯುತ್ತಿರುವ ರೀತಿಯನ್ನು ಅರಿಯೆನು. ತುಂಬಿದ ತಲೆಗೂದಲಿನವರೇ, ಯಾರಾದರೂ ಅರಿಹಾಸ್ಯಮಾಡುವರು ಎಂಬ ಅದರ ಅಪಮಾನವನ್ನು ಬರದಂತೆ ರಕ್ಷಿಸಿಕೊಳ್ಳುವುದಕ್ಕೆ ಆಗದೆ, ಇದ್ದುಕೊಂಡು ಬಾಳನ್ನು ಕಳೆಯುತ್ತಿದ್ದೇನೆ. 



ಇಲ್ಲಿ ಪಾಶುರದ ನಾಯಕಿಯೇ ಮಾತನಾಡುತ್ತಿದ್ದಾಳೆ. ಅವಳಿಗೆ ಆಗಿರುವ ಯಾತನೆಗೆ ನಿಜವಾದ ಕಾರಣವನ್ನೂ, ಅದರಿಂದ ಅವಳು ಉನ್ಮತ್ತಳಾಗಿ, ತನ್ನ ಪ್ರಿಯತಮನ ದಿವ್ಯನಾಮಗಳನ್ನು ಒದರುತ್ತಾ ಅಲೆದಾಡುತ್ತಿರುವುದನ್ನೂ, ತನ್ನನ್ನು ಕಂಡ ಹೆಂಗಳೆಲ್ಲರೂ ಅವಳನ್ನು ಪರಿಹಾಸ್ಯಮಾಡಿ, ನಿಂದಿಸುತ್ತಿರುವುದನ್ನೂ, ತನಗೆ ಅದನ್ನು ಎದುರಿಸಲು ಆಗದಂತೆತಾನು ಕೃಶವಾಗಿ ತನ್ನ ಬಾಳ್ವೆಯನ್ನು ನಡೆಸುತ್ತಿರುವುದನ್ನೂ ಇಲ್ಲಿ ಹೇಳಿಕೊಳ್ಳುತ್ತಾಳೆ.



ಅವಳಿಗೆ ಕಾರ್ಮುಗಿಲನ್ನು ಹೋಲುವ ದಿವ್ಯಸುಂದರ ರೂಪವನ್ನು ಕಂಡಾಗಿನಿಂದಲೂ ಅವಳು ಉನ್ಮತ್ತಲಾಗಿದ್ದಾಳೆ. ಅವನ ಹೆಸರನ್ನೇ ಹೇಳಿಕೊಳ್ಳುತ್ತಾ ಅಲೆದಾಡುತ್ತಿದ್ದಾಳೆ. ಜೊತೆಗೆ ತಂಗಾಳಿಯು ಮೃದುವಾಗಿ ಅವಳ ಮೈಯನ್ನು ಹೊಕ್ಕು, ಅವಳನ್ನು ಕೃಶಗೊಳಿಸಿ ಸಂಕಟಗೊಳಿಸುತ್ತದೆ. ಇದನ್ನುಕಂಡ ಸ್ತ್ರೀಯರೆಲ್ಲರೂ ಬಗೆಬಗೆಯಾಗಿ ಪರಿಹಾಸ್ಯಮಾಡುತ್ತಾರೆ. ಅವರ ನಿಂದನೆಯನ್ನು ಎದುರಿಸಲು ಸಾಧ್ಯವಾಗದೆ, ತನ್ನನ್ನು ರಕ್ಷಿಸಿಕೊಳ್ಳಲು ಆಗದೆ, ಬಾಳ್ವೆಯನ್ನು ಹಾಗೆಯೇ ನಡೆಸುತ್ತಿದ್ದಾಳೆ.

ವಾರಾಯ್ = ಬಾರಯ್ಯ, ಮಡನೆಂಜಮೇ = ತಿಳಿವಳಿಕೆಯಿಲ್ಲದ ಮನಸ್ಸೇ, ವನ್ದು = ಬಂದು, ಮಣಿವಣ್ಣನ್ = ಇಂದ್ರನೀಲಮಣಿಯ ಬಣ್ಣದವನ, ಶೀರ್ ಆರ್ = ಪರಮಮಂಗಳಕರವಾದ, ತಿರುತುಳಾಯ್ = ಪವಿತ್ರವಾದ ತುಲಸಿಯ, ಮಾಲೈ = ಹಾರವನ್ನು, ನಮ್ಮಕ್ಕು ಅರುಳಿ = ನಮಗೆ \(ನನಗೆ\) ಕೃಪೆಮಾಡಿ, ತಾರಾನ್ ತಾರಾನ್ ತರುಮ್ = ತಾರನೇ ತರುವನೇ, ಎನ್ಱು = ಎಂದು, ಇರಣ್ಡತ್ತಿಲ್ = ಎರಡರಲ್ಲಿ, ಒನ್ ಱದನೈ = ಒಂದನ್ನು, ಆರಾನುಮ್ = ಯಾರಾದರೂ, ಒನ್ನಾದಾರ್ = ಇಷ್ಟಪಡದವರು \(ಹೊಂದಿರದವರು\), ಕೇಳಾಮೇ = ಕೇಳದಂತೆ, ಶೊನ್ನಕ್ಕಾಲ್ = \(ನೀನು\) ಹೇಳಿದರೆ, ಆರಾಯುಮ್ = ಪರಿಶೀಲಿಸಿದರೂ \(ಪರೀಕ್ಷಿಸಿ ನೋಡಿದರೂ\), ಪಣಿಕೇಟ್ಟು = ಆ ಮಾತನ್ನು ಕೇಳಿ, ಅದು ಅನ್ ಱು ಎನಿಲುಮ್ = ಅದು ಸರಿಯಲ್ಲ ಎಂದರೂ, ಪೋರಾದು = ಇಲ್ಲಿಗೆ ಬರದಂತೆ \(ಅವರೊಡನೆ ಜಗಳವಾಡದೆ\), ಒೞಿಯಾದೇ = ಅಲ್ಲೆ ಉಳಿಯದೆ \(ಅದರಿಂದಲೇ ಅವರಿಂದಲೇ ನಾಶವಾಗದೆ\), ಪೋನ್ದಿಡು ನೀ = ನೀನು ಇಲ್ಲಿಗೆ ಬಂದು ಬಿಡು, ಎನ್ ಱೇ ಱ್ ಕು= ಎನ್ನುವ ನನಗೆ, ಕಾರ್ ಆರ್ = ಕಾರ್ಮುಗಿಲನ್ನು ಹೋಲುವ, ಕಡಲ್ ವಣ್ಣನ್= ಕಡಲವಣ್ಣದವನನ್ನು, ಪಿನ್ ಪೋನ = ಹಿಂಬಾಲಿಸಿ ಹೋದ, ನೆಂಜಮುಮ್ = ಮನಸ್ಸೂ, ಸಹ, ವಾರಾದೇ = ನನ್ನಲ್ಲಿಗೆ ಬರದಂತೆ, ಎನ್ನೈ = ನನ್ನನ್ನು, ಮುಱನ್ದದು = ಮರೆಯಿತು, ತಾನ್ =ನಾನು, ವಲ್ ವಿನೈ ಯೇನ್ = ಕ್ರೂರಪಾಪಿಯಾದವಳು, ಊರಾರುಮ್ = ಊರಿನವರು, ಉಹಪ್ಪದೇ = ಹರ್ಷಿಸುವುದೇ ಆಯಿನೇನ್ = ಆಗಿದ್ದೇನೆ ನಾನು. 



“ಅರಿಯದ ಮನಸ್ಸೇ, ಬಾರಯ್ಯ, ಒಂದು, ನೀಲಮಣಿ ವಣ್ನನು ಮಂಗಳಕರವಾದ ಪವಿತ್ರವಾದ ತುಲಸಿಯ ಹಾರವನ್ನು ನಮಗೆ \(ನನಗೆ\) ಕೃಪೆ ಮಾಡಿ ತರುವನೇ ತಾರನೇ ಎಂದು ಎರಡರಲ್ಲಿ ಒಂದನ್ನು ಯಾರಾದರೂ ಇಷ್ಟಪಡದವರು ಹೇಳಿದಂತೆ ಕೇಳದೆ, ನೀನು ಹೇಳಿದರೆ, ಅದನ್ನುಪರೀಕ್ಷಿಸಿ ನೋಡಿದರೂ ಸರಿ, ಅದು ಸರಿಯಲ್ಲ ಎಂದರೂ ಸರಿ, ನೀನು ಅವರಲ್ಲಿ ತರ್ಕಮಾಡದೆ, ಅದರಲ್ಲೇ ಲೀನವಾಗದೆ, ನೀನು ಇಲ್ಲಿಗೆ ಬಂದು ಬಿಡು” ಎಂದು ಹೇಳುವ ನನಗೆ, ಕಾರ್ಮುಗಿಲನ್ನು ಹೋಲುವ ಕಡಲವಣ್ಣನನ್ನು ಹಿಂಬಾಲಿಸಿ ಹೋದ ಮನಸ್ಸೂ ಸಹ ನನ್ನಲ್ಲಿಗೆ ಬರದಂತೆ ನನ್ನನ್ನು ಮರೆಯಿತು. ನಾನು ಕಡುಪಾಪಿಯಾದವಳು. ಊರಿನವರು ಹರ್ಷಿಸುವಂತೆಯೇ ಆಗಿದ್ದೇನೆ. 



ಪಾಶುರದ ನಾಯಕಿ ನೊಂದು ನುಡಿಯುತ್ತಿದ್ದಾಳೆ: “ಏನೂ ಅರಿಯದ ನನ್ನ ಮನಸ್ಸೇ, ನೀನು ನೀಲಮಣಿವಣ್ಣದವನ ಬಿಳಿಯ ಪವಿತ್ರವಾದ ತುಲಸಿಯ ಹಾರವನ್ನು ನನಗೆ ಕೃಪೆಮಾಡುವನೇ, ಇಲ್ಲವೇ ಎಂಬುದನ್ನು ನಿಖರವಾಗಿ ತಿಳಿದುಕೊಂಡು ಬಾ. ಯಾರಾದರೂ ಅವನಲ್ಲಿ ಅನುರಕ್ತರಾಗದವರು ಹೇಳಿದ ಮಾತನ್ನು ನೀನು ಪರೀಕ್ಷಿಸಿ ನೋಡಿದರೂ ಸರಿ. ಅವರೊಡನೆ ನೀನು ತರ್ಕಕ್ಕೆ ಇಳಿಯದೆ, ಅದರಲ್ಲೇ ಮಗ್ನವಾಗಿ ಬಿಡದೆ, ನನ್ನಲ್ಲಿಗೆ ಬಂದು ಬಿಡು” – ಎಂದು ನಾನು ಹೇಳಿದೆ. ಆದರೆ ನನ್ನ ಮನಸ್ಸು ಕಡಲವಣ್ಣದವನೂ ಕಾರ್ಮುಗಿಲನ್ನು ಹೋಲುವವನೂ ಆದವನನ್ನು ಹಿಂಬಾಲಿಸಿ ಹೋದದ್ದು ನನ್ನ ಬಳಿಗೆ ಬರಲು ಇಲ್ಲ. ನನ್ನನ್ನು ಪೂರ್ತಿಯಾಗಿ ಮರೆತುಬಿಟ್ಟಿತು. ನಾನು ಕಡುಪಾಪಿ. ಊರಿನವರು \(ನನ್ನನ್ನು ಕಂಡು, ಅಪಹಾಸ್ಯಮಾಡುತ್ತಾ\) ಹರ್ಷಿಸುವಂತೆ ಆಗಿದ್ದೇನೆ.

ಮಟ್ರು = ಬೇರೆ, ಎನಕ್ಕು = ನನ್ನನ್ನು \(ಕುರಿತು\), ಇಙ್ಗು = ಇಲ್ಲಿ, ಆರಾಯ್ ವಾರ್ = ವ್ಯಥೆಪಡುವವರು, ಇಲ್ಲೈ = ಇಲ್ಲ, ಅೞಲ್ ವಾಯ್ = ಬೆಂಕಿಯಲ್ಲಿ, ಮೆೞುಹುಪೋಲ್ = ಅರಗಿನ ಹಾಗೆ, ನೀರಾಯ್ = ನೀರಾಗಿ, ಉರುಹುಮ್ = ಕರಗುತ್ತದೆ, ಎನಾವಿ = ನನ್ನ ಪ್ರಾಣವು, ನೆಡುಕಣ್ ಹಳ್ = ದೊಡ್ಡ ಕಣ್ಣುಗಳು, ಊರಾರ್ = ಊರಿನವರು, ಉಱಙ್ಗಿಲುಮ್ = ನಿದ್ರಿಸಿದರೂ, ತಾಮ್ = ತಾವು, ಉಱಙ್ಗಾ = ನಿದ್ರಿಸದೆ, ಉತ್ತಮನ್ ತನ್ = ಪುರುಷೋತ್ತಮನ, ಪೇರ್ =ಹೆಸರುಗಳು, ಆಯಿನವೇ = ಆಗಿರುವುದನ್ನೇ, ಪಿದಟ್ರುವನ್ = ಒದರುತ್ತಿರುವೆನು, ಪ್ನ್ನೈಯುಮ್ = ಅಲ್ಲದೆ, ಕಾರ್‍ ಆರ್ = ಕಾರ್ಮುಗಿಲನ್ನು ಹೋಲುವ, ಕಡಲ್ ಪೋಲುಮ್ = ಕಡಲನ್ನು ಹೋಲುವ, ಕಾಮತ್ತಾರ್ = ಆಕರ್ಷಕನಾಗಿ, ಆಯಿನಾರ್ = ಆಗಿರುವವನನ್ನು, ಆರೇ = ಯಾರೇ ಪೊಲ್ಲಾಮೈ = ಕೆಟ್ಟವನನ್ನಾಗಿ, ಆಱಿವಾರ್ = ಅರಿತುಕೊಳ್ಳುವರು? 



ಇಲ್ಲಿ ನನ್ನನ್ನು ಕುರಿತು ಬೇರೆ ಯಾರೂ ವ್ಯಥೆಪಡುವವರು ಇಲ್ಲ. ಬೆರಿಕೆಯಲ್ಲಿನ ಅರಗಿನ ಹಾಗೆ ನೀರಾಗಿ ನನ್ನ ಪ್ರಾಣ ಕರಗುತ್ತಿದೆ. ಊರಿನವರೆಲ್ಲರೂ ನಿದ್ರಿಸಿದರೂ ನನ್ನ ದೊಡ್ಡ ಕಣ್ಣುಗಳು ಮಾತ್ರ ನಿದ್ರಿಸವು. ಪುರುಷೋತ್ತಮನ ಹೆಸರುಗಳಾಗಿರುವುದನ್ನೇ ನಾನು ಒದರುತ್ತಿರುವೆನು. ಮತ್ತು \(ಅಲ್ಲದೆ\) ಕಾರ್ಮುಗಿಲನ್ನು ಹೋಲುವ ಕಡಲನ್ನು ಹೋಲುವ ಆಕರ್ಷಕನಾಗಿರುವವನನ್ನು ಕೆಟ್ಟವನನ್ನಾಗಿ ಅರಿತುಕೊಳ್ಳುವವರು ಯಾರು? 



ಪಾಶುರದ ನಾಯಕಿ ಮುಂದುವರಿಸುತ್ತಿದ್ದಾಳೆ: ಈ ಊರಿನಲ್ಲಿ \(ನೆಲದ ಮೇಲೆ\) ಎಲ್ಲರೂ ನನ್ನನ್ನು ಕುರಿತು ಪರಿಹಾಸ್ಯವನ್ನೂ ನಿಂದನೆಯನ್ನೂ ಮಾಡುತ್ತಾರೆಯೇ ವಿನಾ ನನ್ನ ಸಂಕಟವೇನೆಂದು ವಾಸ್ತವವಾಗಿ ಅರಿತುಕೊಂಡು ವ್ಯಥೆಪಡುವವರಿಲ್ಲ. ನನ್ನ ಪ್ರಾಣಗಳು ಬೆಂಕಿಯಲ್ಲಿಟ್ಟ ಅರಗಿನ ಹಾಗೆ ಕರಗಿ ನೀರಾಗುತ್ತಿದೆ. ನನಗೆ ನಿದ್ದೆಯೇ ಹತ್ತುವುದಿಲ್ಲ. ಊರಿನವರೆಲ್ಲರೂ ನಿದ್ರಿಸಿದರೂ, ನನ್ನ ದೊಡ್ಡ ಕಣ್ಣುಗಳಿಗೆ ನಿದ್ದೆ ಬರುವುದಿಲ್ಲ. ಆದ್ದರಿಂದ, ಎಡೆಬಿಡದಂತೆ ನಾನು ಪುರುಷೋತ್ತಮನ ಹೆಸರುಗಳನ್ನು ವಟಗುಟ್ಟುತ್ತಾ ರಾತ್ರಿ ಹಗಲು ಕಾಲಕಳೆಯುವೆನು. ಅತ್ಯಾಕರ್ಷಕನಾದ ಪುರುಷೋತ್ತಮನನ್ನು ಯಾರಾದರೂ ಕೆಟ್ಟವನು ಎನ್ನುತ್ತಾರೆಯೇ? ಅವನು ನನ್ನಲ್ಲಿ ಕನಿಕರ ತೋರಿಸದೆ ಇರುವನೇ?

ತಾರಾರ್ ತಡನ್ತೋಳ್ ತಳೈಕ್ಕಾಲನ್ ಪಿನ್ ಪೋನಾಳ್, 



೬೮. ಊರಾರಿಹೞ್ ನ್ದಿಡಪ್ಪಟ್ಟಾಳೇ, ಮಟ್ರೆನಕ್ಕಿಙ್ಗು

ಆರಾನುಙ್ಕಱ್ ಪಿಪ್ಪಾರ್ ನಾಯಹರೇ.................



ಅದು ನಿಱ್ ಕ = ಅದು ಹಾಗಿರಲಿ, ಆರಾನುಮ್ = ಯಾರಾದರೂ, ಆದಾನುಮ್ = ಏನಾದರೂ, ಅಲ್ಲಳ್ ಅವಳ್ = ಅವಳು ಅಲ್ಲ, ಕಾಣೀರ್ = ಕಾಣೀರಿ, ವಾರ್ ಆರ್ = ಕುಪ್ಪಸ ಬಿಗಿದ, ವನಮ್ ಮುಲೈ = ಸುಂದರವಾದ ಮೊಲೆಗಳ, ವಾಶವದತ್ತೈ ಎನ್ ಱು = ವಾಸವದತ್ತೆ ಎಂಬ, ಆರಾನುಮ್ = ಯಾರಾದರೂ, ಶೊಲ್ಲಪ್ಪಡುವಾಳ್ = ಹೇಳಲ್ಪಡುವವಳು, ಅವಳುಮ್ ತನ್ = ಅವಳೂ ಸಹ ತನ್ನ \(ಅವಳ\), ಪೇರ್ ಆಯಮ್ ಎಲ್ಲಾಮ್ = ಮುಖ್ಯವಾದ ಸಂಗಾತಿಗಳನ್ನೆಲ್ಲಾ, ಒೞಿಯ = ಬಿಟ್ಟು, ಪೆರುನ್ತೆರುವೇ = ದೊಡ್ಡ \(ವಿಶಾಲವಾದ\) ಬೀದಿಯಲ್ಲಿಯೇ, ತಾರ್ ಆರ್ = ಹಾರವನ್ನು ಧರಿಸಿರುವ, ತಡ ತೋಳ್ = ದೀರ್ಘವಾದ ಕೈಗಳ \(ತೋಳುಗಳ\), ತಳೈ ಕಾಲನ್ = ಶಕ್ತಿಪೂರ್ಣವಾದ ಕಾಲುಗಳುಳ್ಳವನ, ಪಿನ್ = ಹಿಂದೆ, ಪೋನಾಳ್ = ಹೋದಳು, ಊರಾರ್ = ಊರಿನವರಿಂದ, ಇಹೞ್ ನ್ದಿಡಪ್ಪಟ್ಟಾಳೇ =ನಿಂದಿಸಲ್ಪಟ್ಟಳೇ? ಮಟ್ರು = ಮತ್ತೆ \(ಆದರೆ\) ಎನಕ್ಕು = ನನಗೆ, ಇಙ್ಗು = ಇಲ್ಲಿ, ಆರಾನುಮ್ = ಯಾರಾದರೂ, ಕಱ್ ಪಿಪ್ಪಾರ್ = ಉಪದೇಶ ಮಾಡುವವರು, ನಾಯಹರೇ = ನಾಯಕರಿದ್ದಾರೆಯೇ? 



ಅದು ಹಾಗಿರಲಿ. ಅವಳು ಯಾರೋ ಏನೋ ಒಬ್ಬಳಲ್ಲ ಕಾಣಿರಿ. ಕುಪ್ಪಸ ಬಿಗಿದ ಸುಂದರವಾದ ಮೊಲೆಗಳ ವಾಸವದತ್ತೆ ಎಂದು ಹೇಳಲ್ಪಡುವವಳು. ತನ್ನ ಮುಖ್ಯ \(ಆತ್ಮೀಯ\) ಸಂಗಾತಿಗಳನ್ನೆಲ್ಲಾ ತೊರೆದು, ವಿಶಾಲವಾದ ಬೀದಿಯಲ್ಲಿಯೇ, ಹಾರವನ್ನು ಧರಿಸಿರುವ ಉದ್ದವಾದ ತೋಳುಗಳೂ, ಶಕ್ತಿಪೂರ್ಣವಾದ ಕಾಲುಗಳೂ ಉಳ್ಳವನ ಹಿಂದೆ ಹೋದಳು. \(ಅವಳು\) ಊರಿನವರಿಂದ ನಿಂದಿಸಲ್ಪಟ್ಟಳೇ? ಆದರೆ, ನನಗೆ, ಇಲ್ಲಿ, ಯಾರಾದರೂ ಉಪದೇಶಮಾಡುವ ನಾಯಕರಿದ್ದಾರೆಯೇ? 



ಇಲ್ಲಿ ಪಾಶುರದ ನಾಯಕಿ ತನಗೆ ತಕ್ಕಂಥ ಒಂದು ಯುಕ್ತವಾದ ನಿದರ್ಶನವನ್ನು ಕೊಡುತ್ತಿದ್ದಾಳೆ. ಅಲ್ಲದೆ, ತನಗೂ ಅವಳಿಗೂ ಇರುವ ವ್ಯತ್ಯಾಸವನ್ನು ಹೇಳುತ್ತಾಳೆ. 



ನಿದರ್ಶನದಲ್ಲಿ ’ವಾಸವದತ್ತೆ’ ಎಂಬುವಳು ಒಬ್ಬಳು. ಅವಳು ದಿವ್ಯಸುಂದರಿ. ’ವತ್ಸರಾಜ’ ಎಂಬವನನ್ನು ಪ್ರೇಮಿಸಿದಳು. ಅವನಿಗಾಗಿ, ತನ್ನ ಆತ್ಮೀಯರಾದ ಗೆಳತಿಯರೆಲ್ಲರನ್ನೂ ತೊರೆದಳು. ಅವನ ಹಿಂದೆಯೇ, ಅವಳು ವಿಶಾಲವಾದ ಬೀದಿಯಲ್ಲಿಯೇ ಎಲ್ಲರೂ ನೋಡುತ್ತಿದ್ದ ಹಾಗೆಯೇ ಹೊರಟುಹೋದಳು. ಅವಳನ್ನು ಯಾರೂ ಹಾಸ್ಯಮಾಡಲಿಲ್ಲ. 



ನಾಯಕಿ ಹೇಳುತ್ತಾಳೆ: ವಾಸವದತ್ತೆ ತನ್ನ ಪ್ರಿಯತಮನನ್ನು ಹಿಂಬಾಲಿಸಿ, ನಡೆದು ಹೋದಾಗ ಯಾರೂ ಅವಳನ್ನು ನಿಂದಿಸಲಿಲ್ಲ. ಅಪಹಾಸ್ಯಮಾಡಲಿಲ್ಲ. ಆದರೆ, ನನ್ನ ಗತಿ ನೋಡಿ. ನನ್ನ ನಲ್ಲನು ನನ್ನಲ್ಲಿಗೆ ಬಂದು ನನ್ನನ್ನು ಕರೆದೊಯ್ಯಲಿಲ್ಲ. ನನ್ನನ್ನು ವೃಥಾಸೊರಗಿಸುತ್ತಿದ್ದಾನೆ. ಅವನನ್ನು ಕೂಡಿಕೊಳ್ಳುವ ರೀತಿಯನ್ನು ಹೇಳಿಕೊಡುವ ನಾಯಕನು ಯಾರಾದರೂ ನನಗೆ ಸಿಕ್ಕುವರೇ? 



ವಾಸವದತ್ತೆ ಎಂಬ ಸುಂದರ ಕನ್ಯೆಯು ವತ್ಸರಾಜನೆಂಬವನನ್ನು ಮೋಹಿಸಿ, ಅವಳ ಆತ್ಮೀಯರಾದ ಗೆಳತಿಯರನ್ನು ತೊರೆದು, ಅವನ ಹಿಂದೆ ಹೊರಟು ಹೋದಳೆಂಬುದು ಒಂದು ಕತೆ. ಆಳ್ವಾರರು ಇದನ್ನು ಇಲ್ಲಿ ಒಂದು ನಿದರ್ಶನವಾಗಿ ಮಾತ್ರವೆ ತೆಗೆದುಕೊಂಡಿರುವುದು ಮೋಹಿತರಾದವರು ಪರಸ್ಪರ ಕೂಡಿ ಒಂದೆಡೆಯಿಂದ ತೆರಳಿದರೆ, ಅವರ ಬಗೆಗೆ, ಅದರಲ್ಲೂ ಹೆಣ್ಣಿನ ದಿಟ್ಟತನದ ಬಗೆಗೆ ಊರಿನ ಜನ ನಾನಾಬಗೆಯಾಗಿ ಆಡಿಕೊಳ್ಳುವುದು ಸಾಮಾನ್ಯ. ಆದರೆ, ವಾಸವದತ್ತೆ ತನ್ನ ಪ್ರಿಯತಮನೊಡನೆ ಎಲ್ಲರೂ ಕಂಡ ಹಾಗೆಯೇ ಹೋದಾಗ, ಅವಳನ್ನು ಯಾರೂ ನಿಂದಿಸಲಿಲ್ಲವಂತೆ. ಪರಿಹಾಸ್ಯ ಮಾಡಲಿಲ್ಲವಂತೆ. ಹೀಗೆ ಪಾಶುರದ ನಾಯಕಿಗೆ \(ಆಳ್ವಾರರಿಗೆ\) ಆಗಲಿಲ್ಲವಲ್ಲ ಎಂಬುದು ಮಾತ್ರವೇ ಇಲ್ಲಿನ ಹೋಲಿಕೆ.

ನಾನ್ =ನಾನು, ಅವನೈ = ಅವನ, ಕಾರ್ ಆರ್ = ಕಾರ್ಮುಗಿಲನ್ನು ಹೋಲುವ, ತಿರುಮೇನಿ = ದಿವ್ಯರೂಪವನ್ನು, ಕಾಣುಮ್ ಅಳವುಮ್ = ಕಾಣುವವರೆಗೂ, ಪೋಯ್ = ಹೋಗಿ, ಶೀರ್ ಆರ್ = ಐಶ್ವರ್ಯ\(ಸೌಂದರ್ಯ\) ತುಂಬಿದ, ತಿರುವೇಙ್ಗಡಮೇ = ಶ್ರೀವೆಂಕಟಾಚಲ, ತಿರುಕ್ಕೋವಲ್ ಊರೇ = ತಿರುಕ್ಕೋವಲೂರು, ಮದಿಳ್ ಕಚ್ಚಿ = ಪ್ರಾಕಾರ\(ಕೋಟೆ\)ದಿಂದ, ಸುತ್ತುವರಿದಿರುವ ಕಾಂಚೀಪುರ, ಊರಾಹಮೇ = ಊರಾಹಮ್, ಪೇರ್ ಆಹಮೇ = ತಿರುಪ್ಪೇರೂರು, ಪೇರಾ = ಪ್ರಸಿದ್ಧವಾದ, ಮರುದು ಇರುತ್ತಾನ್ = ಮತ್ತೀಮರಗಳನ್ನು ಮುರಿದವನ, ವೆಳ್ಳಱೈಯೇ = ತಿರುವೆಳ್ಳರೈ, ವೆಹ್ ಕಾವೇ= ತಿರುವೆಹ್ ಕಾ ಊರೆ, ಪೇರ್ ಆೞ = ದೊಡ್ಡ ಆಲದಮರದ ’ಪೇರಾಲಿ’, \(ತಿರುವಾಲಿನಗರ\), ಕ್ತಣ್ ಕಾಲ್ = ತಿರುತ್ತಣ್ ಕಾಲ್, ನಱೈಯೂರ್ = ತಿರುನರೈಯೂರು, ತಿರುಪ್ಪುಲಿಯೂರ್ = ತಿರುಪ್ಪುಲಿಯೂರ್, ಶೀರಾರ್ = ಸುಂದರವಾದ, ಕಣಪುರಂ ಚೇಱೈ= ಕಣ್ಣಪುರ ಮತ್ತು ಶೇರೈನಗರ, ತಿರುವಱುನ್ದೂರ್ = ತಿರುವಳುನ್ದೂರು, ಕಾರ್ ಆರ್ ಕುಡನ್ದೈ = ಸೊಬಗು ತುಂಬಿದ ಕುಂಭಕೋಣ \(ತಿರುಕ್ಕುಡನ್ದೈನಗರ\), ಕಡಿಹೈ = ತಿರುಕ್ಕಡಿಹೈನಗರ, ಕಡಲ್ ಮಲ್ಲೈ = ಕಡಲ್ ಮಲ್ಲೈ, ಏರ್ ಆರ್ = ಸುಂದರವಾದ, ಪೊೞಿಲ್ = ತೋಟಗಳಿಂದ, ಶೂೞ್ = ಸುತ್ತುವರಿದಿರುವ, ಎನ್ದೈ = ನಮ್ಮ ಸ್ವಾಮಿಯ, ನೀರ್ ಮಲೈ = ತಿರುನೀರ್ ಮಲೈ, ಶೀರ್ ಆರುಮ್ ಮಾಲಿರುಂಜೋಲೈ = ಸುಂದರವಾದ ತಿರುಮಾಲಿರುಂಜೋಲೈ, ತಿರುಮೋಹೂರ್ = ತಿರುಮೋಹೂರು, ಪಾರೋರ್ = ಭೂಲೋಕದವರು, ಪುಹೞುಮ್ = ಹೊಗಳುವ, ವದರಿ = ಬದರಿ, ವಡಮದುರೈ = ಉತ್ತರದೇಶದ ಮಧುರೆ, ಊರ್ ಆಯ = ಮುಂತಾದ ಊರುಗಳಾಗಿರುವ, ಎಲ್ಲಾಮ್ = ಎಲ್ಲವನ್ನೂ, ಒೞಿಯಾಮೇ = ಬಿಡದಂತೆ, ನಾಳ್ = ಯಾವಾಗಲೂ \(ದಿನದಿನವೂ\), ಅವನೈ = ಅವನನ್ನು, ಓರ್ ಆನೈ ಕೊಮ್ಬು ಒಶಿತ್ತು = ಒಂದು ಆನೆಯ ದಂತವನ್ನು ಮುರಿದು, ಓರ್ ಆನೈಕೋಳ್ ವಿಡುತ್ತ = ಒಂದು ಆನೆಯ ಗೋಳನ್ನು ಬಿಡಿಸಿದ, ಶೀರಾನೈ = ಸರ್ವಶ್ರೇಷ್ಠನನ್ನು, ಶೆಮ್ ಕಣ್ ನೆಡಿಯಾನೈ = ಸುಂದರವಾದ ಕಣ್ಣುಗಳ ಸರ್ವೇಶ್ವರನನ್ನು, ತೇನ್ ತುೞಾಯ್ ತಾರಾನಿ = ಜೇನು ತುಂಬಿದ ತುಲಸಿಯಹಾರವನ್ನು ಧರಿಸಿರುವವನನ್ನು, ತಾಮರೈ ಪೋಲ್ ಕಣ್ಣಾನೈ = ತಾವರೆಯಂತೆ ಕಣ್ಣುಳ್ಳವನನ್ನು, ಎಣ್ = ಎಣಿಸಲಾಗದ, ಅರು ಶೀರ್ = ಅತಿ ದಿವ್ಯವಾದ, ಪೇರ್ = ಹೆಸರುಗಳು, ಆಯಿರಮುಮ್ = ಸಾವಿರವನ್ನೂ, ಪಿದಟ್ರಿ = ಒದರುತ್ತಾ, ಪೆರುತೆರುವೇ = ದೊಡ್ಡ ಬೀದಿಯಲ್ಲಿ \(ಉದ್ದಕ್ಕೂ\), ಊರಾರ್ = ಊರಿನವರು, ಇಹೞಿಲುಮ್ = ತೆಗಳಿದರೂ \(ನಿಂದಿಸಿದರೂ\), ಊರಾದು = ಹರಡುವುದನ್ನು, ಒೞಿಯೇನ್ = ಬಿಡೆನು, ನಾನ್ = ನಾನು, ವಾರ್ ಆರ್ = ಬಲು ಉದ್ದನಾದ, ಪೂ= ಸುಂದರವಾದ, ಪೆಣ್ಣೈ = ತಾಳೆಯ ಗರಿಯಲ್ಲಿ, ಮಡಲ್ = \(ನನ್ನ\) ಪ್ರೇಮಗೀತೆಯನ್ನು. 



ಇಲ್ಲಿ ಪಾಶುರದ ನಾಯಕಿಯೇ ಮಾತನಾಡುತ್ತಿದ್ದಾಳೆ. ಅವಳಿಗೆ ಉಂಟಾದ ನೋವಿಗೆ ಕಾರಣವನ್ನೂ, ಅದರಿಂದ ಅವಳೇಕೆ ಉನ್ಮತ್ತಳಾಗಿ ಅಲೆದಾಡುತ್ತಿದ್ದಳೆಂದೂ, ಕಂಡಕಂಡ ಹೆಂಗಳೆಲ್ಲರೂ ಅವಳನ್ನು ಅರಿತುಕೊಳ್ಳದೆಯೇ ಅವಳನ್ನು ನಿಂದಿಸುವುದನ್ನೂ, ಪರಿಹಾಸ್ಯಮಾಡುವುದನ್ನೂ, ಅವುಗಳನ್ನೆದುರಿಸಲು ಅವಳಿಗೆ ಸಾಧ್ಯವಾಗದೆ, ಅವಳ ಸಂಕಟದಿಂದಲೇ ಕೃಶನಾಗುತ್ತಿದ್ದಾಳೆಂದೂ ಹೇಳಿಕೊಳ್ಳುತ್ತಿದ್ದಾಳೆ. 



ಭಾರತನಾಡಿನ ನಾನಾಪವಿತ್ರ ಕ್ಷೇತ್ರಗಳನ್ನು ಇಲ್ಲಿ ಎತ್ತಿಕೊಳ್ಳಲಾಗಿದೆ. ನೂರ ಎಂಟು ತಿರುಪತಿಗಳು ಭಾರತದಲ್ಲಿ ಸುಪ್ರಸಿದ್ಧವಾಗಿವೆಯೆಂದೂ, ಅವುಗಳನ್ನು ಭಕ್ತನಾದವನು ತಪ್ಪದೆ ಸಂದರ್ಶಿಸಬೇಕೆಂದೂ, ಆಯಾ ಸ್ಥಳದಲ್ಲಿ ಭಗವಂತನು ನೆಲಸಿರುವುದಕ್ಕೆ ಇರುವ ವೈಶಿಷ್ಟ್ಯವೇನೆಂಬುದನ್ನು ಕಂಡುಕೊಳ್ಳಬೇಕೆಂದೂ ಇದರಿಂದ ಭಕ್ತಿ ಸ್ಥಿರಗೊಳ್ಳುವುದೆಂದೂ ಹೇಳಲಾಗಿದೆ. ಅಲ್ಲದೆ, ಭಗವಂತನ ಲೀಲಾವಿಭೂತಿಗಳನ್ನೂ, ಅವನ ಗುಣಸ್ವಭಾವಗಳನ್ನೂ ಎಡೆಬಿಡದಂತೆ ಚಿಂತಿಸುತ್ತಿರಬೇಕೆಂದೂ, ಅವನ ಪವಿತ್ರವಾದ ಸಹಸ್ರನಾಮಗಳನ್ನು ಸದಾ ಹೇಳಿಕೊಳ್ಳುತ್ತಿರಬೇಕೆಂದೂ ಹೇಳಲಾಗಿದೆ. ಭಗವಂತನ ವಿಷಯದಲ್ಲಿ ಉಂಟಾಗುವ ಅತಿಶಯವಾದ ಪ್ರೇಮ ಫಲವಾದ ಈ ಗೀತೆಯು ಸ್ವಾರಸ್ಯವೇ ಇದು ಎನಿಸುತ್ತದೆ. 



“ಓರಾನೈಕೊಮ್ಬೊಶಿತ್ತು” – ಇದು ಭಗವಂತನ ಶ್ರೀಕೃಷ್ಣಾವತಾರದ ಒಂದು ಪ್ರಸಂಗ. ತನ್ನ ಕಡು ಶತ್ರುವಾದ ಬಾಲಕೃಷ್ಣನನ್ನು ಕೊಲ್ಲಿಸಲು ಸಜ್ಜುಗೊಳಿಸಿ, ಅವನನ್ನು ಧನುರ್ಯಾಗದ ನೆಪದಿಂದ ಮಧುರೆಗೆ ಕಂಸನು ಕರೆಸಿದನು. ಮಧುರೆಯ ಹೆಬ್ಬಾಗಿಲಲ್ಲೇ ಕುವಲಯಾಪೀಡವೆಂಬ ಮದ್ದಾನೆ ಅವನನ್ನು ತುಳಿದು ಕೊಲ್ಲಲು ಕಾದಿತ್ತು. ಬಾಲಕೃಷ್ಣನು ಅದನ್ನೆದುರಿಸಿ, ಅದರ ಒಂದು ದಂತವನ್ನೇ ಮುರಿದುಕೊಂಡು, ಅದರಿಂದಲೇ ಅದನ್ನು ಕೊಂದು ಹಾಕಿದನು ಎಂಬುದು ಕತೆ. 



’ಓರಾನೈಕೋಳ್ ವಿಡುತ್ತು’ – ಭಗವಂತನ ಗಜೇಂದ್ರಮೋಕ್ಷದ ಪ್ರಸಂಗವಿದು. ಸರೋವರದಲ್ಲಿ ನೀರು ಕುಡಿಯಲು ಇಳಿದ ಮದ್ದಾನೆಯೊಂದರ ಕಾಲನ್ನು ಮೊಸಳೆಯೊಂದು ಹಿಡಿದುಕೊಂಡಿತು. ಅದರೊಡನೆ ಸೆಣಸಿ ಸೆಣಸಿ, ಆನೆ ಹಣ್ಣಾಯಿತು. ಗತ್ಯಂತರವಿಲ್ಲದೆ, ಅದು ಭಗವಂತನನ್ನು ದೈನ್ಯವಾಗಿ ಪ್ರಾರ್ಥಿಸಿತು. ಭಗವಂತನು ಕೂಡಲೇ ಅದರ ಬಳಿಗೆ ಧಾವಿಸಿ ಬಂದು, ತನ್ನ ಚಕ್ರಾಯುಧದಿಂದ ಮೊಸಳೆಯನ್ನು ಸಂಹರಿಸಿ, ಆನೆಯನ್ನು ಸಂಕಟದಿಂದ ಬಿಡಿಸಿದನು. 



ನಾಯಕಿ ಹೇಳುತ್ತಾಳೆ: ನಾನು ಅವನ ದಿವ್ಯರೂಪವನ್ನು ಕಾಣುವವರೆಗೂ ಸುಪ್ರಸಿದ್ಧವಾದ ನಾನಾ ದಿವ್ಯಕ್ಷೇತ್ರಗಳನ್ನು – ಎಂದರೆ, ತಿರುವೆಂಕಟಾದ್ರಿ, ತಿರುಕ್ಕೋವಲೂರು, ಕಂಚಿ, ಊರಾಹ, ತಿರುಪ್ಪೇರೂರು, ತಿರುವೆಳ್ಳರೈ, ತಿರುವೆಹ್ ಕಾ, ತಿರುವಾಲಿನಗರ, ತಿರುತ್ತಣ್ ಕಾಲ್, ತಿರುನರೈಯೂರು, ತಿರುಪ್ಪುಲಿಯೂರು, ತಿರುಕ್ಕಣ್ಣಪುರ, ಶೇರೈನಗರ, ತಿರುವಳುನ್ದೂರು, ಕುಂಭಕೋಣ, ತಿರುಕ್ಕಡಿಹೈ, ತಿರುನೀರ್ ಮಲೈ, ತಿರುಮಾಲಿರುಂಜೋಲೈ, ತಿರುಮೋಹೂರು, ಬದರಿ, ಉತ್ತರ ಮಧುರೆ ಮುಂತಾದ ದಿವ್ಯಕ್ಷೇತ್ರಗಳನ್ನೆಲ್ಲಾ ಬಿಡದಂತೆ ಸಂದರ್ಶಿಸುತ್ತಿರುವೆನು. ದಿವ್ಯವಾದ ಅವನ ಸಾವಿರ ನಾಮಗಳನ್ನು ಬಿಡದೆ ಒದರುತ್ತಿರುವೆನು. ದುಷ್ಟಶಿಕ್ಷಕನೂ, ಶಿಷ್ಟರಕ್ಷಕನೂ, ವಿಶಾಲವೂ ಸುಂದರವೂ ಆದ ಕಣ್ಣುಳ್ಳವನೂ, ಪರಿಮಳ ತುಂಬಿದ ತುಲಸಿಯ ಹಾರವನ್ನು ಧರಿಸಿದವನೂ ಆದ ಅವನ ಸಾವಿರನಾಮಗಳನ್ನು ಒದರುತ್ತಾ, ಊರಿನವರೆಲ್ಲರೂ ತೆಗಳಿದರೂ, ನಿಂದಿಸಿದರೂ, ನನ್ನ ಈ ಪ್ರೇಮಗೀತೆಯನ್ನು ಉದ್ದನಾದ ತಾಳೆಯಗರಿಯಲ್ಲಿ ಬರೆದಿಟ್ಟು, ಅದನ್ನು ಎಲ್ಲಲ್ಲೂ ಹರಡದೆ ಬಿಡುವುದಿಲ್ಲ. 



ಎಂಧ ಸದ್ದುದ್ದೇಶ\! ತಾವು ಮಾತ್ರವೇ ಉದ್ಧಾರಗೊಳ್ಳುವುದಕ್ಕೆ ಬದಲಾಗಿ, ಎಲ್ಲರೂ ಉದ್ಧಾರಗೊಳ್ಳಲೆಂದು ಮಾಡುವ ಈ ಪ್ರಯತ್ನ ಎಷ್ಟು ಪವಿತ್ರವಾದದ್ದು\!

ತಿರುವಾಳನ್ ಶೀರ್ ಕ್ಕಲಿಯನ್ ಕಾರ್ ಕ್ಕಲಿಯೈ ವೆಟ್ಟಿ,

ಮರುವಾಳನ್ ತನ್ದಾನ್ ಮಡಲ್.



ಮುಳ್ಳಿ = \(ಮುಳ್ಳಿನಿಂದ ಕೂಡಿದ ಪರಿಮಳದ ಹೂವಿನ\) ತಾಳೆಯ ಹೂವಿನ, ಶೆೞು = ಸುಂದರವಾದ, ಮಲರ್ = ಹೂವಿನ, ಓರ್ = ಅಪರೂಪವಾದ, ತಾರಾನ್ = ಹಾರವನ್ನು ಧರಿಸಿರುವವನು, ಮುಳೈ = ಮೊಳೆಯುತ್ತಿರುವ, ಮದಿಯನ್ = ಚಂದ್ರನ, ಕೊಳ್ಳಿಕ್ಕೂ = ಬೇಗೆಗೆ, ಎನ್ ಉಳ್ಳಮ್ = ನನ್ನ ಮನಸ್ಸು, ಕೊದಿಯಾಮೇ = ಕುದಿಯಂತೆ ಇರುವುದಕ್ಕೆಂದು, ವೆಳ್ಳಲ್ = ಉದಾರಿಯೂ, ತಿರುವಾಳನ್ = \(ಭಗವದ್ಗುಣಾನುಭವೆಂಬ\) ಐಶ್ವರ್ಯವನ್ನುಅನುಭವಿಸುವವನೂ, ಶೀರ್ ಕಲಿಯನ್ = ಶ್ರೇಷ್ಠನಾದ ಕಲಿಯನು, \(ತಿರುಮಂಗೈ ಆಳ್ವಾರರು\), ಕಾರ್ ಕಲಿಯೈ = ಕ್ರೂರವಾದ ಅಜ್ಞಾನವೆಂಬ ದೋಷವನ್ನು, ವೆಟ್ಟಿ = ಕತ್ತರಿಸತಕ್ಕ, ಮರುವಾಳನ್ = ತಕ್ಕ ಆಯುಧವನ್ನುಳ್ಳವನು \(ಕತ್ತಿಯನ್ನುಳ್ಳವನು\), ತನ್ದಾನ್ = ಕೊಟ್ಟನು, ಮಡಲ್ = ತಿರುಮಡಲ್ ಎಂಬ ಪ್ರೇಮಗೀತೆಯನ್ನು.



ತಾಳೆಯ ಗಿಡದ ಸುಂದರವಾದ ಹೂವಿನ ಅಪರೂಪವಾದ ಹಾರವನ್ನು ಧರಿಸಿರುವವನೂ, ಉದಾರಿಯೂ, ಭಗವದ್ಗುಣಾನುಭವವೆಂಬ ಐಶ್ವರ್ಯವನ್ನು ಅನುಭವಿಸತಕ್ಕವನೂ, ಶ್ರೇಷ್ಠನಾದ ಕಲಿಯನೂ, ಕ್ರೂರವಾದಜ್ಞಾನವೆಂಬ \(ಅಜ್ಞಾನಾಂಧಕಾಶವೆಂಬ\) ದೋಷವನ್ನು ಕತ್ತರಿಸತಕ್ಕ ಒಳ್ಳೆಯ ಆಯುಧವುಳ್ಳವನೂ ಆದ ತಿರುಮಂಗೈಆಳ್ವಾರರು. ಉದಯಿಸುತ್ತಿರುವ \(ಮೊಳೆಯುತ್ತಿರುವ\) ಚಂದ್ರನ ಬೇಗೆಗೆ ನನ್ನ ಮನಸ್ಸು ಕುದಿಯದಂತೆ ಇರುವುದಕ್ಕೆಂದು ತಿರುಮಡಲ್ ಎಂಬ ಪ್ರೇಮಗೀತೆಯನ್ನು ಅನುಗ್ರಹಿಸಿದನು.



ತಿರುಮಂಗೈಆಳ್ವಾರರನ್ನು ಪ್ರಶಂಶಿಸುವ ತನಿ ಇದು. ಈ ಆಳ್ವಾರನ್ನು ’ಕಲಿಯನ್’ ’ಕಲಿಕನ್ ಱಿ’ ’ಕಲಿಧ್ವಂಸಿ’ ಎಂಬ ಹೆಸರುಗಳಿಂದ ಸಂಬೋಧಿಸುವುದು ವಾಡಿಕೆ. ಸಾಕ್ಷಾತ್ ಶ್ರೀಮನ್ನಾರಾಯಣನಿಂದಲೇ ನೇರವಾಗಿ ಮಂತ್ರೋಪದೇಶವನ್ನು ಪಡೆದು, ಮಹಾಜ್ಞಾನಿಯಾಗಿ, ಭಗವದ್ಗುಣಾನುಭವವನ್ನು ಅನುಭವಿಸುತ್ತಾ, ಆ ಮಹದೈಶ್ವರ್ಯವನ್ನು ಸವಿಯುತ್ತಾ ಕಲಿಯುಗದ ಅಜ್ಞಾನವೆಂಬ ಗಾಢಾಂಧಕಾರನೆಂಬ ದೋಷವನ್ನು ಕತ್ತರಿಸಿ, ಕಲಿಯುಗದ ಜನರನ್ನು ಮೋಕ್ಷಮಾರ್ಗಕ್ಕೆ ಎಳೆಯುವುದಕ್ಕಾಗಿ, ವಿಪುಲವಾಗಿ ದಿವ್ಯಪ್ರಬಂಧಗಳನ್ನು ರಚಿಸಿದ ಮಹಾನುಭಾವನು ಈತ. ಸಾಮಾನ್ಯವಾಗಿ ಈತನ ಕೈಯಲ್ಲಿ ಮೇಲಾಯುಧವನ್ನೂ, ಕೊರಳಲ್ಲಿ ಪಾದರಿಹೂವಿನ ಮಾಲಿಕೆಯನ್ನೂ ಧರಿಸಿರುವುದು ವಾಡಿಕೆಯೆಂದಾದರೂ, ಈ ತನಿಯಲ್ಲಿ ಕೇದಿಗೆ ಹೂವಿನ \(ಮುಳ್ಳಿನ ಪರಿಮಳದ ಹೂವಿನ\) ಮಾಲಿಕೆಯನ್ನು ಧರಿಸಿದ್ದಾನೆಂದೂ, ಕೈಯಲ್ಲಿ ಕತ್ತಿಯನ್ನು ಹಿಡಿದಿದ್ದಾನೆಂದೂ ವಿವರಿಸಲಾಗಿದೆ. ಕಲಿಯುಗದ ಅಜ್ಞಾನವನ್ನು ಕತ್ತರಿಸಿಹಾಕಲು ಕತ್ತಿಯೇ ಬೇಕೇನೋ\!



ತನಿ ಹೇಳುತ್ತದೆ: ನನ್ನಲ್ಲಿರುವ ಅಜ್ಞಾನವೆಂಬ ಕಗ್ಗತ್ತಲೆಯನ್ನು ನೀಗಿಸುವುದಕ್ಕೂ, ಮೊಳೆಯುತ್ತಿರುವ \(ಬಾಲ\) ಚಂದ್ರನ ಬೇಗೆಯನ್ನು ನಿವಾರಿಸುವುದಕ್ಕೂ ’ತಿರುಮಡಲ್’ ಎಂಬ ಪ್ರೇಮಗೀತೆಯನ್ನು ತಿರುಮಂಗೈ ಆಳ್ವಾರರು ನನಗೆ ಔದಾರ್ಯದಿಂದ ಅನುಗ್ರಹಿಸಿದ್ದಾರೆ.
